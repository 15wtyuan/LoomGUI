# v1.4-a 包加载模型重构 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把 v1 错误耦合的包加载模型（包=单树、load=建 scene、图集在核心）重构为 fgui/Unity prefab 式干净模型——包=多组件模板库、load_package 只进资源池、instantiate 克隆组件挂 layer、核心只知图片 path 图集归 Unity Sprite Atlas，为 v1.4-b 虚拟化列表（item 模板=instantiate）铺地基。

**Architecture:** pkg.bin 从"单树 + AtlasSection + root_size"改为"多组件 ComponentTable + 分组件 dynamic_rules + AssetManifest + 砍 atlas/root_size"；`Stage` 加 `packages: HashMap<String, Package>` 资源池，`load_package(name, bytes)` 不碰 scene 只填资源池，`instantiate(pkg, comp) -> NodeId` 克隆 ComponentTemplate 子树进 scene（复用 v1.3+ create_node）+ 伪类规则合并去重；Image payload 从 tex_id/UV 改带 path（FrameBlob 字符串表）；Unity 侧 path→Sprite 缓存查 Sprite Atlas，砍 LoadAtlas/_texMap；Unity 包管理面板（ScriptableObject + EditorWindow）；showcase 重做多页架构。

**Tech Stack:** Rust 2021 / slotmap 1.1.1 / taffy 0.5 / csbindgen / Unity 6.5 URP（C# 后端 + Sprite Atlas + EditorWindow）。

## Global Constraints

- **spec 权威**：本 plan 实现 specs/2026-07-02-v1.4a-package-loading-design.md。16 条决策（D1-D16）逐条落地，不得偏离。spec 与 plan 冲突时以 spec 为准，发现 spec 错则停下报告。
- **包=多组件**：一个 pkg.bin = 一个系统 = 多个 HTML 文件，每个 HTML = 一个具名组件，组件名 = 文件名去 `.html`（D1/D2）。不支持组件间静态引用（D5）。
- **load_package 不碰 scene**：`Stage::load_package(name, bytes)` 只填 `self.packages[name]`，scene 完全不变（D3）。`Stage::instantiate(pkg, comp) -> NodeId` 克隆子树返回孤立根，调用方 append_child 挂载（D4）。
- **核心不知图集**：Image 节点 payload 带 path（res 前缀后的相对路径），无 atlas/tex_id/UV 概念。图集完全归 Unity Sprite Atlas（D8/D14）。
- **CSS 组件隔离**：每 HTML 自带 CSS（`<style>`/`<link>`/行内均可），打包器归一化 bake 进节点 style_blob，组件间不污染（D6）。伪类规则组件级存，instantiate 合并去重进 scene（D7）。
- **root_size 归 Stage**：pkg.bin header 砍 root_w/root_h（D9）。res 目录名可配置（默认 `res`，D10）。
- **旧 pkg 不兼容**：bump PKG_FORMAT_VERSION，旧 pkg read 报错，showcase 等全部重打，无迁移器（D11）。
- **砍 load_inline**：移除 `Stage::load_inline`，全走 package 路径，单测改用内存 pkg（D12）。
- **复用 v1.3+ 动态树**：instantiate 内部复用 `scene/dynamic.rs` 的 create_node 逻辑 + `append_child`。代际 NodeId + slotmap + remove_node 联动清不变量保留。
- **零回归**：渲染等价（旧 showcase 场景用新流程跑，几何/样式/图片/事件/滚动/动画/伪类不变）。fence_contract 围栏测试全过。
- **两台机串行**：本机唯一编码机（Rust build .dll + 打包新格式 pkg + commit + push），家里机纯 Unity PlayMode 验收。改 Rust 后必重编 + commit .dll 家里机才能测。
- **语言**：问答/总结中文，代码/commit 英文。

- [ ] **Step 5: 重写 write_package**

新格式（照 spec §3.2）：
```
Header: magic + version + flags + component_count + string_count
StringTable: 组件名/text content/img path/classes/id_attr intern 去重
ComponentTable: 每组件 {name_idx(u16), root_node_idx(u32), node_count(u32), dynamic_rules_blob_len(u32)}
NodeBlock: 所有组件节点平铺，每节点 {parent_idx(i32,-1=根), kind_tag(u8), style_blob, text_idx(u16), src_idx(u16),
          class_count(u16)+class_idx[], id_idx(u16), flags(u8), tabindex(i32)}
PerComponentDynamicRules: 每组件 dynamic_rules 的 bincode blob（紧跟 ComponentTable 后，按组件序）
AssetManifest: path_count(u32) + path_idx[](u16)
```
- `parent_idx`：模板内全局位置（组件根=-1）。多组件 NodeBlock 拼接，ComponentTable.root_node_idx 切分。
- `src_idx` 指向 path 字符串（img），不再是 atlas sprite。
- 砍 root_w/root_h（header 28B→20B）。

```rust
/// 归一化图片 src：去 res 目录前缀 + 统一正斜杠。
/// 不在 res 下 → None（调用方 warning）。res_dir 默认 "res"（可配置）。
pub fn normalize_path(src: &str, res_dir: &str) -> Option<String> {
    let unified = src.replace('\\', "/");                    // Win 反斜杠 → 正斜杠
    let prefix = format!("{}/", res_dir);
    // 找 res/ 前缀（允许前面有 ./ 或 /）
    let stripped = unified
        .find(&prefix)
        .map(|i| &unified[i + prefix.len()..])
        .filter(|s| !s.is_empty());
    // 确认 res/ 是路径段（不是 "pres/" 误匹配）：检查前缀前一个字符是 / 或开头或 .
    stripped.map(String::from)
}
```

```rust
/// 从包克隆一个组件进当前 scene，返回组件根 NodeId（孤立，调用方 append_child 挂载）。
pub fn instantiate(&mut self, pkg: &str, component: &str) -> Result<NodeId, String> {
    let scene = self.scene.as_mut().ok_or("no scene (create_root first)")?;
    let template = self.packages.get(pkg)
        .and_then(|p| p.components.get(component))
        .ok_or_else(|| format!("component {} not in pkg {}", component, pkg))?
        .clone();   // clone 出 borrow，避开 packages/scene 双借用

    // 遍历 template.nodes，按 parent_idx 序建 live Node（父先建于子）
    // 复用 dynamic.rs 的 create_node 逻辑（kind/style → base_style + style 初始/classes/id_attr/...）
    let mut id_map: Vec<Option<NodeId>> = vec![None; template.nodes.len()]; // 模板 idx → live NodeId
    for (i, tn) in template.nodes.iter().enumerate() {
        let node_id = scene_insert_node(scene, tn);   // 建 Node 进 slotmap，填 base_style/style/classes/id_attr
        id_map[i] = Some(node_id);
        if let Some(pidx) = tn.parent_idx {
            let parent = id_map[pidx].expect("parent built before child");
            // 串 children（scene.nodes.get_mut(parent).children.push(node_id) + node.parent = parent）
        }
        // 根（parent_idx=None）不串父，记录返回
    }
    let root = id_map[template.root_node_idx?].expect("root built");

    // 伪类规则合并去重
    for rule in &template.dynamic_rules.rules {
        if !scene.dynamic_rules.rules.iter().any(|r| r.selector == rule.selector) {
            scene.dynamic_rules.rules.push(rule.clone());
        }
    }
    Ok(root)
}
```
**注意**：`scene_insert_node` 是把 TemplateNode 建成 live Node 的辅助（复用 dynamic.rs create_node 的节点构造部分，但跳过 CSS parse——style 已 bake 在 TemplateNode）。root_node_idx 从 ComponentTemplate 取（Task 1 read 时填，或在此算 parent_idx=None 的节点）。

```rust
pub extern "C" fn loomgui_stage_load_package(
    h: *mut StageHandle, name: *const u8, name_len: usize, bytes: *const u8, bytes_len: usize,
) -> i32 {
    let sh = unsafe { &mut *(h) };
    let name = unsafe { std::slice::from_raw_parts(name, name_len) };
    let name = std::str::from_utf8(name).unwrap_or("");
    let bytes = unsafe { std::slice::from_raw_parts(bytes, bytes_len) };
    match sh.stage.load_package(name, bytes) { Ok(()) => 0, Err(_) => 1 }
}

pub extern "C" fn loomgui_stage_instantiate(
    h: *mut StageHandle, pkg: *const u8, pkg_len: usize, comp: *const u8, comp_len: usize,
) -> u32 {
    let sh = unsafe { &mut *(h) };
    let pkg = unsafe { std::str::from_utf8_unchecked(std::slice::from_raw_parts(pkg, pkg_len)) };
    let comp = unsafe { std::str::from_utf8_unchecked(std::slice::from_raw_parts(comp, comp_len)) };
    match sh.stage.instantiate(pkg, comp) {
        Ok(id) => id.0,
        Err(_) => 0xFFFF_FFFF,   // INVALID
    }
}
```

- [ ] **Step 1: 重写 driver 结构**

```csharp
class LoomShowcaseDriver : MonoBehaviour {
    LoomStage _stage;
    uint _root, _uiLayer, _tipsLayer;
    uint _currentPage = uint.MaxValue;   // 当前页根 NodeId（home 初始）
    string _currentPkg = "showcase";
    // 当前页 listener 注册表：nodeId → [(eventType, callback)]，切页前清
    Dictionary<uint, List<(EventType, EventCallback)>> _pageListeners = new();

    void Start() {
        _root = _stage.CreateRoot("div", "...");
        _uiLayer = _stage.CreateNode("div", "...");   _stage.AppendChild(_root, _uiLayer);
        _tipsLayer = _stage.CreateNode("div", "..."); _stage.AppendChild(_root, _tipsLayer);
        _stage.LoadPackage("showcase", LoadBytes("loom_showcase.pkg.bin"));
        OpenPage("home");
    }

    void OpenPage(string page) {
        if (_currentPage != uint.MaxValue) {
            ClearPageListeners();              // 按页清 listener（§7.5）
            _stage.RemoveNode(_currentPage);   // 摘当前页（联动清 anim/scroll/tween/focused_node）
        }
        _currentPage = _stage.Instantiate("showcase", page);
        _stage.AppendChild(_uiLayer, _currentPage);
        SubscribePage(page);                   // 按页订阅（home→导航；各 page→其交互元素）
    }
}
```

- [ ] **Step 2: 按页订阅（SubscribePage）**

`SubscribePage(page)` switch page → 调 `SubscribeHome`/`SubscribeControls`/.../`SubscribeDynTree`（从当前 driver 的 SubscribeLampEvents/SubscribeTweenDemos/SubscribeDynamicTree 抽，每页一组）。每订阅记进 `_pageListeners`（而非直接 AddListener）：
```csharp
void AddPageListener(uint node, EventType t, EventCallback cb) {
    _stage.EventHandler.AddListener(node, t, cb);
    _pageListeners.GetOrAdd(node).Add((t, cb));
}
void ClearPageListeners() {
    foreach (var (node, list) in _pageListeners)
        foreach (var (t, cb) in list) _stage.EventHandler.RemoveListener(node, t, cb);
    _pageListeners.Clear();
}
```
