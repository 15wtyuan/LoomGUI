# v1d.5 ScrollPane 实现计划（12 任务 TDD）

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现 ScrollPane（overflow:scroll/auto + 双轴+轴锁 + 惯性/回弹自带 tween + 滚动条含 grip 拖拽 + 滚轮 + 单层/嵌套仲裁），关 v1 验收 #2 可滚动容器。

**Architecture:** `scroll_pos` 作虚拟平移注入 `compute_world_transforms` DFS（子节点 world 前乘 `T(-父.scroll_pos)`，容器自身/viewport/clip 固定）——零 DOM 改、复用 v1d.3 world matrix + 现有 clip 基建。物理自维护可变 target tween（**不走 GTween**），fgui 直译。仲裁 per-slot（scrolling_pane/dragging_node 互斥）。

**Tech Stack:** Rust core（loomgui_core + loomgui_ffi_c）+ Unity 6.5 C# 后端 + csbindgen FFI。

## Global Constraints

- **用户只读中文**：Q&A/选项/总结一律中文；**代码/commit 英文**。
- **两机工作流**：本机唯一编码机（Rust `cargo test` + build .dll + commit + push）；家里机纯 Unity PlayMode 验收。**改 Rust FFI/blob 后必重编 .dll + commit**（坑 10），家里机才能测。C# 本机不编译（无 Unity）。
- **main 直推**（与 v1d.2/.3/.4 一致）。
- **Ponytail full**：最短 diff、YAGNI、无不请求的抽象。

- **scroll 不走 GTween**：自维护 tween，API 层无 GTween-tween-scroll_pos 入口（避免双写）。

- **FFI enum 必须 `#[repr(u8)]`**（坑 34）；`size_of::<T>()` 断言 ABI struct 尺寸。

```rust
//! v1d.5 ScrollPane 状态 + 物理（spec §2/§4）。transient（不进 pkg）。

use crate::scene::node::{NodeId, Scene};
use crate::style::resolved::OverflowMode;

#[derive(Debug, Clone, Copy, PartialEq, Default)]
pub struct ScrollPaneState {
    pub content_size: (f32, f32),
    pub viewport_size: (f32, f32),
    pub overlap: (f32, f32),
    pub scroll_pos: (f32, f32),
    pub velocity: (f32, f32),
    pub tweening: u8,           // 0=无 1=set_pos 2=惯性+回弹
    pub tween_start: (f32, f32),
    pub tween_change: (f32, f32),
    pub tween_time: (f32, f32),
    pub tween_duration: (f32, f32),
    pub content_size_dirty: bool,
}

#[derive(Debug, Clone, Default)]
pub struct ScrollTable(pub Vec<Option<ScrollPaneState>>);

impl ScrollTable {
    pub fn get(&self, id: NodeId) -> Option<&ScrollPaneState> {
        self.0.get(id.0).and_then(|o| o.as_ref())
    }
    pub fn get_mut(&mut self, id: NodeId) -> Option<&mut ScrollPaneState> {
        self.0.get_mut(id.0).and_then(|o| o.as_mut())
    }
    pub fn ensure(&mut self, id: NodeId) -> &mut ScrollPaneState {
        if id.0 >= self.0.len() { self.0.resize(id.0 + 1, None); }
        self.0[id.0].get_or_insert_with(ScrollPaneState::default)
    }
    pub fn clear(&mut self) { self.0.clear(); }
}

/// 该轴是否允许滚动（overflow ∈ {Scroll, Auto}）。
pub fn capable(node_overflow: OverflowMode) -> bool {
    matches!(node_overflow, OverflowMode::Scroll | OverflowMode::Auto)
}

/// 该轴实际可滚（capable 且 (Scroll 或 content>viewport)）。
pub fn effective(ovf: OverflowMode, content: f32, viewport: f32) -> bool {
    capable(ovf) && (ovf == OverflowMode::Scroll || content > viewport)
}

/// solve 后填 content_size/viewport/overlap（spec §2.3）。遍历滚动容器（任一轴 ≠ Visible）。
pub fn refresh_content_sizes(scene: &mut Scene) {
    let ids: Vec<usize> = (0..scene.nodes.len()).collect();
    for id in ids {
        let nid = NodeId(id);
        let is_scroll = {
            let n = &scene.nodes[id];
            n.style.overflow_x != OverflowMode::Visible || n.style.overflow_y != OverflowMode::Visible
        };
        if !is_scroll { continue; }
        // content_size = 直接子节点 layout_rect AABB
        let kids = scene.nodes[id].children.clone();
        let (mut min_x, mut min_y) = (f32::MAX, f32::MAX);
        let (mut max_x, mut max_y) = (f32::MIN, f32::MIN);
        for c in &kids {
            let r = scene.nodes[c.0].layout_rect;
            min_x = min_x.min(r.x); min_y = min_y.min(r.y);
            max_x = max_x.max(r.x + r.w); max_y = max_y.max(r.y + r.h);
        }
        let content = if kids.is_empty() {
            (0.0, 0.0)
        } else {
            ((max_x - min_x).max(0.0), (max_y - min_y).max(0.0))
        };
        let n = &scene.nodes[id];
        let viewport = content_box_size(n);  // content box（见下 helper）
        let st = scene.scroll.ensure(nid);
        st.content_size_dirty = st.content_size != content;
        st.content_size = content;
        st.viewport_size = viewport;
        st.overlap = ((content.0 - viewport.0).max(0.0), (content.1 - viewport.1).max(0.0));
    }
}

/// content box 尺寸 = layout_rect 尺寸减 padding（四边）减 border_width。
/// implementer 核实 ResolvedStyle 的 padding/border 字段名（grep `padding\|border_width` in resolved.rs）。
fn content_box_size(node: &crate::scene::node::Node) -> (f32, f32) {
    let lr = node.layout_rect;
    // v1 简化：scroll 容器建议 padding:0；读 style padding/border 减去。
    // 若 ResolvedStyle 暴露 padding(top/right/bottom/left) + border_width：
    //   (lr.w - pad_l - pad_r - 2*border, lr.h - pad_t - pad_b - 2*border)
    // 字段名以实际为准；padding:0/border:0 时退化为 (lr.w, lr.h)。
    (lr.w, lr.h)  // ponytail: v1 先用 border box；padding 边缘处理 defer（spec §1.3 已认）
}
```

> **注意**：`content_box_size` v1 用 border box（layout_rect 尺寸）。spec §1.3 已声明 padding 简化（建议 scroll 容器 padding:0）。若 implementer 发现 padding 字段易取，可改为 content box——非阻塞。

`loomgui_core/src/scroll.rs` 加 `impl ScrollPaneState`（照 spec §4.2，fgui 直译）：
```rust
impl ScrollPaneState {
    /// 拖拽跟手：scroll_pos += delta（越界 PULL_RATIO 打折）+ 记速度。
    pub fn drag_follow(&mut self, delta: (f32, f32), dt: f32) {
        // 速度记录（exp 平滑）
        if dt > 0.0 {
            let smoothing = (dt * VELOCITY_SMOOTH).clamp(0.0, 1.0);
            self.velocity.0 = self.velocity.0 + (delta.0 / dt - self.velocity.0) * smoothing;
            self.velocity.1 = self.velocity.1 + (delta.1 / dt - self.velocity.1) * smoothing;
        }
        for ax in 0..2 {
            let cur = if ax == 0 { self.scroll_pos.0 } else { self.scroll_pos.1 };
            let d = if ax == 0 { delta.0 } else { delta.1 };
            let ov = if ax == 0 { self.overlap.0 } else { self.overlap.1 };
            let vp = if ax == 0 { self.viewport_size.0 } else { self.viewport_size.1 };
            let mut np = cur + d;
            let lo = 0.0f32; let hi = ov;
            if np < lo {
                let over = (lo - np).min(vp * PULL_RATIO);
                np = lo - over * PULL_RATIO;
            } else if np > hi {
                let over = (np - hi).min(vp * PULL_RATIO);
                np = hi + over * PULL_RATIO;
            }
            if ax == 0 { self.scroll_pos.0 = np; } else { self.scroll_pos.1 = np; }
        }
        self.tweening = 0;  // 拖拽中无 tween
    }

    /// Up 后启惯性（tweening=2）。is_touch 选阈值。
    pub fn begin_inertia(&mut self, is_touch: bool) {
        let thresh = if is_touch { INERTIA_THRESH_TOUCH } else { INERTIA_THRESH_PC };
        for ax in 0..2 {
            let v = if ax == 0 { self.velocity.0 } else { self.velocity.1 };
            let ov = if ax == 0 { self.overlap.0 } else { self.overlap.1 };
            let v2 = v * v;
            if v2 < thresh || ov <= 0.0 {
                // 该轴无惯性；若越界启回弹
                continue;
            }
            let dur = (std::f32::consts::60.0 / v2).log(DECELERATION_RATE).abs() / 60.0;
            let dur = dur.max(TWEEN_TIME_DEFAULT);
            let change = v * dur * INERTIA_DIST_COEFF;
            let start = if ax == 0 { self.scroll_pos.0 } else { self.scroll_pos.1 };
            if ax == 0 { self.tween_start.0 = start; self.tween_change.0 = change; self.tween_duration.0 = dur; self.tween_time.0 = 0.0; }
            else { self.tween_start.1 = start; self.tween_change.1 = change; self.tween_duration.1 = dur; self.tween_time.1 = 0.0; }
        }
        self.tweening = 2;
    }

    /// 回弹 tween（越界 > BOUNCE_THRESHOLD）。
    pub fn begin_bounce(&mut self) {
        for ax in 0..2 {
            let cur = if ax == 0 { self.scroll_pos.0 } else { self.scroll_pos.1 };
            let ov = if ax == 0 { self.overlap.0 } else { self.overlap.1 };
            let boundary = if cur < 0.0 { 0.0 } else if cur > ov { ov } else { continue; };
            if (cur - boundary).abs() < BOUNCE_THRESHOLD { continue; }
            let start = cur; let change = boundary - cur;
            if ax == 0 { self.tween_start.0 = start; self.tween_change.0 = change; self.tween_duration.0 = TWEEN_TIME_DEFAULT; self.tween_time.0 = 0.0; }
            else { self.tween_start.1 = start; self.tween_change.1 = change; self.tween_duration.1 = TWEEN_TIME_DEFAULT; self.tween_time.1 = 0.0; }
        }
        self.tweening = 2;
    }

    /// 推进 tween（tweening≠0）。完成 → clamp + 检查回弹 + 归零。
    pub fn advance(&mut self, dt: f32) {
        if self.tweening == 0 { return; }
        for ax in 0..2 {
            let dur = if ax == 0 { self.tween_duration.0 } else { self.tween_duration.1 };
            if dur <= 0.0 { continue; }
            let t = if ax == 0 { &mut self.tween_time.0 } else { &mut self.tween_time.1 };
            *t += dt;
            let norm = (*t / dur).min(1.0);
            let e = cubic_out(norm);
            let new_pos = if ax == 0 {
                self.tween_start.0 + self.tween_change.0 * e
            } else {
                self.tween_start.1 + self.tween_change.1 * e
            };
            if ax == 0 { self.scroll_pos.0 = new_pos; } else { self.scroll_pos.1 = new_pos; }
        }
        // 完成判定（两轴 time ≥ duration）
        let done = (self.tween_duration.0 <= 0.0 || self.tween_time.0 >= self.tween_duration.0)
                && (self.tween_duration.1 <= 0.0 || self.tween_time.1 >= self.tween_duration.1);
        if done {
            // clamp 到 [0, overlap]
            self.scroll_pos.0 = self.scroll_pos.0.clamp(0.0, self.overlap.0);
            self.scroll_pos.1 = self.scroll_pos.1.clamp(0.0, self.overlap.1);
            // 若仍越界 > 阈值 → 重启回弹；否则停
            let still_over = self.scroll_pos.0 < -BOUNCE_THRESHOLD || self.scroll_pos.0 > self.overlap.0 + BOUNCE_THRESHOLD
                          || self.scroll_pos.1 < -BOUNCE_THRESHOLD || self.scroll_pos.1 > self.overlap.1 + BOUNCE_THRESHOLD;
            if still_over { self.begin_bounce(); } else { self.tweening = 0; }
        }
    }

    /// 滚轮：瞬移 -delta*SCROLL_STEP，越界启回弹，否则启 tween。
    pub fn apply_wheel(&mut self, delta: (f32, f32)) {
        // delta.y > 0 = 上滚（看上方）→ scroll_pos.y 减少
        for ax in 0..2 {
            let d = if ax == 0 { delta.0 } else { delta.1 };
            if d == 0.0 { continue; }
            let cur = if ax == 0 { self.scroll_pos.0 } else { self.scroll_pos.1 };
            let target = (cur - d * SCROLL_STEP).max(0.0);
            let target = if ax == 0 { target.min(self.overlap.0) } else { target.min(self.overlap.1) };
            let start = if ax == 0 { self.scroll_pos.0 } else { self.scroll_pos.1 };
            if ax == 0 { self.tween_start.0 = start; self.tween_change.0 = target - start; self.tween_duration.0 = TWEEN_TIME_DEFAULT; self.tween_time.0 = 0.0; }
            else { self.tween_start.1 = start; self.tween_change.1 = target - start; self.tween_duration.1 = TWEEN_TIME_DEFAULT; self.tween_time.1 = 0.0; }
        }
        self.tweening = 1;
    }

    /// 编程滚动。
    pub fn set_pos(&mut self, target: (f32, f32), animated: bool) {
        if !animated {
            self.scroll_pos = (target.0.clamp(0.0, self.overlap.0), target.1.clamp(0.0, self.overlap.1));
            self.tweening = 0;
            return;
        }
        for ax in 0..2 {
            let t = if ax == 0 { target.0.clamp(0.0, self.overlap.0) } else { target.1.clamp(0.0, self.overlap.1) };
            let start = if ax == 0 { self.scroll_pos.0 } else { self.scroll_pos.1 };
            if ax == 0 { self.tween_start.0 = start; self.tween_change.0 = t - start; self.tween_duration.0 = TWEEN_TIME_DEFAULT; self.tween_time.0 = 0.0; }
            else { self.tween_start.1 = start; self.tween_change.1 = t - start; self.tween_duration.1 = TWEEN_TIME_DEFAULT; self.tween_time.1 = 0.0; }
        }
        self.tweening = 1;
    }
}

fn cubic_out(t: f32) -> f32 {
    // (t-1)^3 + 1，t∈[0,1]
    let u = t - 1.0;
    u * u * u + 1.0
}
```

```rust
/// hit(x,y) → 沿祖先找最近 effective 滚动容器 → apply_wheel。无祖先丢弃。
pub fn apply_wheel_to_hit(scene: &mut Scene, w: WheelEvent) {
    let mut pane = crate::hit::hit_test(scene, (w.x, w.y));
    while let Some(id) = pane {
        if id.0 < scene.nodes.len() {
            let n = &scene.nodes[id.0];
            let eff_y = effective(n.style.overflow_y, scene.scroll.get(id).map_or(0.0,|s|s.content_size.1), scene.scroll.get(id).map_or(0.0,|s|s.viewport_size.1));
            let eff_x = effective(n.style.overflow_x, scene.scroll.get(id).map_or(0.0,|s|s.content_size.0), scene.scroll.get(id).map_or(0.0,|s|s.viewport_size.0));
            if eff_y || eff_x {
                if let Some(s) = scene.scroll.get_mut(id) { s.apply_wheel((w.delta_x, w.delta_y)); }
                return;
            }
        }
        pane = scene.nodes[id.0].parent;
    }
}
```

```rust
pub const V_THUMB_FLAG: u32 = 0x4000_0000;
pub const H_THUMB_FLAG: u32 = 0x2000_0000;

/// 垂直 thumb design-rect（容器 viewport 右边缘 track；thumb 大小/位置）。
pub fn v_thumb_rect(scene: &Scene, id: NodeId) -> Option<crate::scene::node::Rect> {
    let s = scene.scroll.get(id)?;
    if s.overlap.1 <= 0.0 { return None; }
    let lr = scene.nodes[id.0].layout_rect;
    let track_w = 8.0;  // v1 固定宽
    let track_h = lr.h;
    let thumb_h = (s.viewport_size.1 * (s.viewport_size.1 / s.content_size.1)).max(20.0).min(track_h);
    let perc = if s.overlap.1 > 0.0 { (s.scroll_pos.1 / s.overlap.1).clamp(0.0, 1.0) } else { 0.0 };
    let thumb_y = lr.y + (track_h - thumb_h) * perc;
    Some(crate::scene::node::Rect { x: lr.x + lr.w - track_w, y: thumb_y, w: track_w, h: thumb_h })
}
// h_thumb_rect 同理（沿底边）
```
