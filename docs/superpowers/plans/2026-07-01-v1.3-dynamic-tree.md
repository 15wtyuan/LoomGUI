# v1.3+ 动态树重构 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把 v1 的 static-tree pipeline 重构为动态树 pipeline——代际 NodeId + slotmap 存储 + 运行时建/挂/删/改节点 API，为 v1.4 列表虚拟化 + v1.5+ 全部动态需求打地基。

**Architecture:** `NodeId` 从 `pub usize` 改为 `pub u32`（高 20 bit index + 低 12 bit generation，对外仍 u32 不透明句柄，FFI/C# 零变化）；`scene.nodes` 从 `Vec<Node>` 改 `SlotMap<NodeId, Node>`（slotmap 1.1.1 已在依赖树）；持久附属数组 anim/scroll 用 `Vec<Option<T>>` + 删节点联动清；每帧重建数组 world_transforms/text_layouts 保持 Vec；动态 API（create_node/append_child/remove_node/set_text/set_src/set_style）入参 CSS 字符串复用 mapping.rs `apply_decl`。

**Tech Stack:** Rust 2021 / slotmap 1.1.1 / taffy 0.5 / csbindgen / Unity 6.5 URP（C# 后端）。

## Global Constraints

- **NodeId 物理表示**：`pub struct NodeId(pub u32)`，高 20 bit = index（`fn index(&self) -> usize { (self.0 >> 12) as usize }`），低 12 bit = generation。sentinel `0xFFFF_FFFF = NodeId::INVALID`。FFI 全程 `node_id: u32`，签名零变化。
- **节点存储**：`scene.nodes: SlotMap<NodeId, Node>`，用 `slotmap::new_key_type!` 生成 NodeId 为 slotmap Key。slotmap 1.1.1 已在 Cargo.lock（taffy 带），**不新增依赖**。
- **持久附属数组**：anim/scroll 用 slotmap `SecondaryMap<NodeId, T>`，按 NodeId 直接索引（悬空安全 + remove 自带清）。
- **每帧重建数组**：world_transforms/text_layouts 保持 `Vec<T>`，每帧 `vec![default; slotmap_capacity]`，按 `NodeId.index()` 索引，空洞位置 default 无害。
- **动态 API 入参**：create_root/create_node/set_style 接 CSS 声明串，复用 `loomgui_core/src/style/mapping.rs::apply_decl(style, prop, value) -> bool`（默认编译，不依赖 parse feature）。
- **base/style 已分离**：`Node.base_style`（源）+ `Node.style`（派生）字段已存在，`rematch_pseudo_classes` 已从 base_style 起算（dynamic.rs:203）。set_style 改 base_style，下帧 rematch 自动跟进。**本 plan 不改 rematch 逻辑**。
- **零回归**：v1 既有 pkg/inline 加载、渲染、滚动、输入、动画、伪类、showcase 全部不变。既有 70+ 处 `Scene { ... }` 字面量 + `NodeId(0)`/`scene.nodes[0]` 字面量改用 `Scene::build` / test helper。
- **两台机串行**：本机唯一编码机（Rust build .dll + commit + push），家里机纯 Unity PlayMode 验收。改 Rust 后必重编 + commit .dll 家里机才能测。

```rust
/// 不透明节点句柄。对外 u32（FFI/C# 透明），内部 = 高 20 bit index + 低 12 bit generation。
/// sentinel 0xFFFF_FFFF = INVALID。index 用于并行数组（anim/scroll/world_transforms）索引，
/// gen 由 slotmap 校验悬空。详见 v1.3+ 动态树 spec §3。
#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash)]
pub struct NodeId(pub u32);

impl NodeId {
    /// 无效句柄 sentinel（同 v1 FFI None/0 约定）。
    pub const INVALID: NodeId = NodeId(0xFFFF_FFFF);
    pub fn is_valid(self) -> bool { self.0 != 0xFFFF_FFFF }
    /// slotmap 槽位号（高 20 bit）。并行数组按此索引。
    pub fn index(self) -> usize { (self.0 >> 12) as usize }
    /// generation（低 12 bit）。slotmap 内部校验用。
    pub fn gen(self) -> u16 { (self.0 & 0xFFF) as u16 }
}
```

`scene/node.rs` 顶部加：

```rust
use slotmap::{SlotMap, new_key_type};
new_key_type! {
```

**注意**：`new_key_type!` 会重新定义 `NodeId`——与 Task 1 的 `pub struct NodeId(pub u32)` 冲突。Task 1 的手动定义在本任务**删除**，改由 `new_key_type!` 生成。但 `new_key_type!` 生成的 Key 没有 `index()`/`gen()`/`INVALID` 方法——这些要在 `impl NodeId` 块里手动加（Task 1 的 impl 块保留，但 `NodeId::index()` 实现要适配 slotmap Key 的内部表示）。

**slotmap Key 内部表示**：`new_key_type!` 生成 `NodeId`（含 slotmap 内部 `u32` data：高 bit slot index + 低 bit version，具体位宽 slotmap 内部定，1.1 默认 24 bit index + 8 bit version）。**关键**：`NodeId::index()`（Task 1）返回 slot index，用于 world_transforms/text_layouts 这类每帧重建的 Vec 索引（按 slot index 容忍空洞，不需 gen 校验）。**持久附属数组 anim/scroll 用 `SecondaryMap<NodeId, T>`**（Task 3），按 NodeId 直接索引（`Index<K>` 实现），**不需 `index()` 位运算**——消解了"位宽不确定"问题。详见 spec §5。

**Task 1 的手动 `impl NodeId` 与 `new_key_type!` 衔接**：`new_key_type!` 生成的 NodeId 是带 slotmap KeyData 的类型。Task 1 加的 `index()`/`gen()`/`INVALID`/`is_valid()` 方法保留——但 `index()` 实现改为从 slotmap KeyData 取 slot index（slotmap 1.1 KeyData 有 `data: u32`，slot index = `data >> version_bits`；实现期用 slotmap 暴露的方式，或 `key.as_ffi()`）。`INVALID = NodeId(0xFFFF_FFFF)`：实现期验证 slotmap 永不分配此值（slotmap KeyData 全 1 是其内部 sentinel，slotmap 不返回——需断言确认）。

**若 slotmap Key index 提取不便**，fallback：`Scene` 维护 `node_index: SlotMap<NodeId, ()>` 不行……实际最干净是 `scene.nodes: SlotMap<NodeId, Node>` + 并行状态用 `slotmap::SecondaryMap<NodeId, T>`（slotmap 原生，按 Key 索引，零位运算，悬空安全同 slotmap）。**这会简化 Task 3**——但需在 Task 2 决定。

**本步骤产出**：`Scene.nodes: SlotMap<NodeId, Node>`，`Scene::build` 用 `scene.nodes.insert(node)` 返回 NodeId 填 `node.id` + parent/children。`find_by_id_attr` 改 `scene.nodes.iter().find(...)`（slotmap 迭代器 yield `(NodeId, &Node)`）。

```rust
pub struct Scene {
    pub roots: Vec<NodeId>,
    pub nodes: SlotMap<NodeId, Node>,
    // ... 其余不变
}

impl Scene {
    pub fn build(entries: &[(Option<usize>, NodeKind, ResolvedStyle, Vec<String>, Option<String>, bool, Option<i32>)]) -> Scene {
        let mut scene = Scene {
            roots: Vec::new(),
            nodes: SlotMap::with_key(),
            dynamic_rules: Default::default(),
            focused_node: None,
            world_transforms: Vec::new(), anim: Default::default(),
            scroll: Default::default(), text_layouts: Vec::new(),
        };
        // 先 insert 所有节点，收集 slotmap 分配的 NodeId
        let mut ids: Vec<NodeId> = Vec::with_capacity(entries.len());
        for (_, kind, style, classes, id_attr, draggable, tabindex) in entries.iter() {
            let node = Node {
                id: NodeId::INVALID, // 临时，insert 后回填
                parent: None, // 下一轮填
                kind: kind.clone(),
                style: style.clone(),
                base_style: style.clone(),
                taffy_id: None,
                layout_rect: Rect::default(),
                clip_rect: if style.overflow_x != OverflowMode::Visible
                    || style.overflow_y != OverflowMode::Visible { Some(Rect::default()) } else { None },
                children: Vec::new(),
                dirty_mesh: true,
                dirty_text: matches!(kind, NodeKind::Text { .. }),
                classes: classes.clone(),
                id_attr: id_attr.clone(),
                touchable: style.touchable,
                hovered: false, active: false, disabled: false,
                draggable: *draggable, tabindex: *tabindex, focused: false,
            };
            let id = scene.nodes.insert(node);
            scene.nodes.get_mut(id).unwrap().id = id; // 回填
            ids.push(id);
        }
        // 接 parent/children/roots（用 ids 映射 entries 下标 → NodeId）
        for (i, (parent_idx, _, _, _, _, _, _)) in entries.iter().enumerate() {
            match parent_idx {
                Some(p) => {
                    let child_id = ids[i];
                    let parent_id = ids[*p];
                    scene.nodes.get_mut(child_id).unwrap().parent = Some(parent_id);
                    scene.nodes.get_mut(parent_id).unwrap().children.push(child_id);
                }
                None => scene.roots.push(ids[i]),
            }
        }
        scene
    }

    /// test helper：从节点列表 + (parent_idx, child_idx) 边建 Scene。替代 70+ 字面量。
    pub fn from_nodes(nodes: Vec<Node>, edges: Vec<(usize, usize)>) -> Scene {
        let mut scene = Scene { roots: Vec::new(), nodes: SlotMap::with_key(),
            dynamic_rules: Default::default(), focused_node: None,
            world_transforms: Vec::new(), anim: Default::default(),
            scroll: Default::default(), text_layouts: Vec::new() };
        let mut ids: Vec<NodeId> = Vec::with_capacity(nodes.len());
        for mut n in nodes {
            let id = scene.nodes.insert(n);
            scene.nodes.get_mut(id).unwrap().id = id;
            ids.push(id);
        }
        for (p, c) in edges {
            let pid = ids[p]; let cid = ids[c];
            scene.nodes.get_mut(cid).unwrap().parent = Some(pid);
            scene.nodes.get_mut(pid).unwrap().children.push(cid);
        }
        // roots = 无 parent 的
        for &id in &ids {
            if scene.nodes.get(id).unwrap().parent.is_none() { scene.roots.push(id); }
        }
        scene
    }

    pub fn find_by_id_attr(&self, id: &str) -> Option<NodeId> {
        self.nodes.iter().find(|(_, n)| n.id_attr.as_deref() == Some(id)).map(|(nid, _)| nid)
    }
}
```

```rust
use crate::scene::node::{Scene, NodeId};
use crate::tween::TweenManager;

/// 删节点：递归删子 + 联动清 anim/scroll/tween + slotmap remove。
/// 旧 NodeId 此后失效（slotmap gen++，get 返 None）。
pub fn remove_node(scene: &mut Scene, tweens: &mut TweenManager, id: NodeId) {
    // 1. 递归删子（先收集 children，再递归——避免边迭代边改）
    let children: Vec<NodeId> = scene.nodes.get(id).map(|n| n.children.clone()).unwrap_or_default();
    for c in children {
        remove_node(scene, tweens, c);
    }
    // 2. 从父摘除
    if let Some(parent_id) = scene.nodes.get(id).and_then(|n| n.parent) {
        if let Some(p) = scene.nodes.get_mut(parent_id) {
            p.children.retain(|&c| c != id);
        }
    } else {
        // 是根：从 roots 摘
        scene.roots.retain(|&r| r != id);
    }
    // 3. 联动清持久附属 map
    scene.anim.remove(id);
    scene.scroll.remove(id);
    tweens.kill_node(id);
    // 4. slotmap remove（gen++）
    scene.nodes.remove(id);
}
```

```rust
/// tag 字符串 → NodeKind（复用 dom.rs 围栏白名单语义，runtime 可用，不依赖 parse feature）。
pub fn kind_from_tag(tag: &str) -> Result<NodeKind, String> {
    match tag {
        "div" | "l-container" => Ok(NodeKind::Container),
        "button" => Ok(NodeKind::Button),
        "img" => Ok(NodeKind::Image { src: String::new() }),
        "span" => Ok(NodeKind::Text { content: String::new() }),
        other => Err(format!("unknown kind tag: {}（围栏白名单：div/l-container/button/img/span）", other)),
    }
}

/// CSS 声明串（"width:100px;background:#f00"）→ 应用到 ResolvedStyle。
/// 极简分割（split(';') + split(':')），逐条 apply_decl。
pub fn apply_css(style: &mut ResolvedStyle, css: &str) {
    for decl in css.split(';') {
        let decl = decl.trim();
        if decl.is_empty() { continue; }
        if let Some((prop, val)) = decl.split_once(':') {
            apply_decl(style, prop.trim(), val.trim());
        }
    }
}
```

```rust
pub fn create_node(scene: &mut Scene, kind: &str, css: &str) -> Result<NodeId, String> {
    let k = kind_from_tag(kind)?;
    let mut base_style = ResolvedStyle::default();
    apply_css(&mut base_style, css);

        taffy_id: None, layout_rect: crate::scene::node::Rect::default(),
        clip_rect: clip, children: Vec::new(),
        dirty_mesh: true, dirty_text: matches!(k, NodeKind::Text { .. }),
        classes: Vec::new(), id_attr: None, touchable: true,
        hovered: false, active: false, disabled: false,
        draggable: false, tabindex: None, focused: false,
    });

pub fn create_root(scene: &mut Scene, kind: &str, css: &str) -> Result<NodeId, String> {
    let id = create_node(scene, kind, css)?;
    scene.roots.push(id);
    Ok(id)
}

pub fn append_child(scene: &mut Scene, parent: NodeId, child: NodeId) -> Result<(), String> {
    let p = scene.nodes.get(parent).ok_or("parent not live")?;
    if p.children.contains(&child) { return Ok(()); }
    // child 必须当前无父
    if scene.nodes.get(child).and_then(|c| c.parent).is_some() {
        return Err("child already has parent（先 remove_child）".into());
    }
    scene.nodes.get_mut(parent).unwrap().children.push(child);
    scene.nodes.get_mut(child).unwrap().parent = Some(parent);
    Ok(())
}

pub fn insert_before(scene: &mut Scene, parent: NodeId, child: NodeId, ref_id: NodeId) -> Result<(), String> {
    if !ref_id.is_valid() { return append_child(scene, parent, child); }
    if scene.nodes.get(child).and_then(|c| c.parent).is_some() {
        return Err("child already has parent".into());
    }
    let p = scene.nodes.get_mut(parent).ok_or("parent not live")?;
    let pos = p.children.iter().position(|&c| c == ref_id).ok_or("ref not in parent.children")?;
    p.children.insert(pos, child);
    scene.nodes.get_mut(child).unwrap().parent = Some(parent);
    Ok(())
}

pub fn remove_child(scene: &mut Scene, parent: NodeId, child: NodeId) -> Result<(), String> {
    let p = scene.nodes.get_mut(parent).ok_or("parent not live")?;
    p.children.retain(|&c| c != child);
    scene.nodes.get_mut(child).unwrap().parent = None;
    Ok(())
}

pub fn set_text(scene: &mut Scene, node: NodeId, text: &str) -> Result<(), String> {
    let n = scene.nodes.get_mut(node).ok_or("node not live")?;
    match &mut n.kind {
        NodeKind::Text { content } => { *content = text.into(); }
        _ => return Err("set_text 只对 Text 节点".into()),
    }
    n.dirty_text = true;
    Ok(())
}

pub fn set_src(scene: &mut Scene, node: NodeId, src: &str) -> Result<(), String> {
    let n = scene.nodes.get_mut(node).ok_or("node not live")?;
    match &mut n.kind {
        NodeKind::Image { src: s } => { *s = src.into(); }
        _ => return Err("set_src 只对 Image 节点".into()),
    }
    n.dirty_mesh = true;
    Ok(())
}

pub fn set_style(scene: &mut Scene, node: NodeId, css: &str) -> Result<(), String> {
    let n = scene.nodes.get_mut(node).ok_or("node not live")?;

    n.dirty_mesh = true;
```
