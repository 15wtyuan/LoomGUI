# v1c.3 多触摸 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把单指针 PointerState 改成固定 5 槽多触摸（1 鼠标 + 4 触摸），加 CaptureTouch/touch monitor 机制，Move 产事件对齐 fgui（无 monitor 不产），EventRecord/PointerEvent 加 touch_id 贯穿。

**Architecture:** capture 是输入状态（核心持 touch_monitors），不是路由策略。核心多槽状态机产事件（Move/Up 看 monitor 决定产不产、产给谁），C# 业务侧沿 node_parent 链路由（Down/Up/Click BubbleRoute，Move/RollOver/Out DirectDispatch）。方向 A 保留。active/hovered 全局 union recompute（多指合并）。

**Tech Stack:** Rust core（loomgui_core input.rs）+ C ABI（loomgui_ffi_c lib.rs，csbindgen）+ Unity C#（loomgui_unity Runtime）。core TDD 本机闭环；C# 本机不编译（无 Unity），家里机验。

**Spec:** `docs/superpowers/specs/2026-06-23-v1c.3-multi-touch-design.md`

## Global Constraints

- **方向 A**：路由/listener 在 C# 业务侧（LoomEventHandler）；核心只命中 + 命中 diff（hover/active 状态 + RollOver/Out 产出）+ 伪类 rematch + **多槽状态机 + touch_monitors**。
- **EventRecord 破零改**：16→20 字节（+touch_id:i32 @8）。波及：C# `LoomEvent` struct +touch_id、`borrow_events` 消费侧 stride（`Marshal.SizeOf<LoomEvent>()` 自动跟）、ffi_c abi_tests 手解 `rec_size` 16→20、新增 sizeof 断言。`.pkg.bin` v3 **不动**（EventRecord 不进包）。
- **PointerEvent +touch_id:i32 @4**（FFI 内部，16 字节）。所有 PointerEvent 构造点（input.rs 测、stage.rs 测、ffi_c abi_tests、LoomInputCollector）须加 touch_id 字段。
- **固定 5 槽**：slots[0]=鼠标（touch_id=-1 常驻），slots[1..4]=触摸（touch_id=-1 空闲，Down 分配 fingerId）。鼠标+触摸同帧共存。
- **touch_id 哨兵**：鼠标=-1（slots[0] 具体值）。**不实现 fgui -1 广播**（add_touch_monitor 只按具体 touch_id 找槽）。
- **CaptureTouch 照 fgui**：`_touchCapture` 消费即清零；capture 阶段 + bubble 阶段**各加一个** monitor（典型拖拽只 bubble 加 1 个）。
- **Move 语义**：有 monitor 产 Move@monitor 直派（每 monitor 一条）；无 monitor 不产。核心 process 收到 Move 仍更新 last_pos + 跑 hover_diff，只是不产 Move EventRecord。
- **active/hovered 全局 union**：Down/Up 分支**不调** set_active_chain；active/hovered 纯由 process 末尾 recompute 驱动。
- **Up 去重**：核心产 monitor 的 Up 时，若 monitor==hit 不重复产。
- **click 判定沿用 v1c.2**（down_node==hit && <10px），不改。click 增强 defer v1c.4。
- **RollOver/Out per-touch**（每槽独立 hover_diff）+ **hovered 全局 union**（双语义，可能不一致，各描述不同事实）。
- **node_parent FFI / frame blob / MirrorPool / MaterialManager / shader / EventBridge / .pkg.bin v3**：零改。
- **两机串行**：core cargo test 本机自检；C# 家里机 EditMode+PlayMode 验。改 Rust 后必重编+commit .dll（坑10 两机变体）。

#[derive(Debug, Clone)]
pub struct TouchSlot {
    pub touch_id: i32,                  // -1=鼠标主指/空闲触摸槽；>=0=触摸 fingerId
    pub last_pos: (f32, f32),
    pub is_down: bool,
    pub down_node: Option<NodeId>,
    pub down_pos: (f32, f32),
    pub last_hit: Option<NodeId>,       // v1c.3：本帧命中（hover_diff + is_pointer_on_ui 用）
    pub last_hovered_chain: Vec<NodeId>,
    pub touch_monitors: Vec<NodeId>,    // v1c.3：capture 的节点（T2 填派发逻辑）
}

impl TouchSlot {
    fn new_mouse() -> Self {
        Self { touch_id: -1, last_pos: (0.0, 0.0), is_down: false, down_node: None,
            down_pos: (0.0, 0.0), last_hit: None, last_hovered_chain: Vec::new(), touch_monitors: Vec::new() }
    }
    fn new_free() -> Self {
        Self { touch_id: -1, last_pos: (0.0, 0.0), is_down: false, down_node: None,
            down_pos: (0.0, 0.0), last_hit: None, last_hovered_chain: Vec::new(), touch_monitors: Vec::new() }
    }
    fn is_active(&self) -> bool {
        // 鼠标主指恒活跃；触摸槽 touch_id>=0（已分配）活跃
        self.touch_id == -1 && std::ptr::eq(self, self) || self.touch_id >= 0
    }
}

/// 多指针状态机（v1c.3：固定 5 槽）。slots[0]=鼠标，slots[1..4]=触摸。
pub struct PointerState {
    pub slots: Vec<TouchSlot>,
}

impl Default for PointerState {
    fn default() -> Self {
        let mut slots = Vec::with_capacity(5);
        slots.push(TouchSlot::new_mouse());     // slot 0 = 鼠标主指
        for _ in 0..4 { slots.push(TouchSlot::new_free()); }  // slot 1..4 = 触摸
        Self { slots }
    }
}

    /// 任一活跃槽命中非根节点 → UI 挡住（§10.6）。
    pub fn is_pointer_on_ui(&self, scene: &Scene) -> bool {
        let root_id = scene.roots.first().copied();
        for slot in &self.slots {
            if let Some(hit) = slot.last_hit {

    pub fn is_pointer_on_ui(&self, scene: &Scene) -> bool {

    /// 找/分配槽。鼠标(touch_id=-1)恒 slots[0]；触摸按 touch_id 找，找不到且 Down→分配首个空闲。
    /// 返回 slot index；找不到（非 Down 无主 / 触摸槽满）→ None。
    fn find_or_alloc_slot(&mut self, ev: &PointerEvent) -> Option<usize> {
        if ev.touch_id == -1 { return Some(0); }   // 鼠标主指
        // 找已占触摸槽
        for i in 1..self.slots.len() {
            if self.slots[i].touch_id == ev.touch_id { return Some(i); }
        }
        // Down 才分配

                    self.slots[i].touch_id = ev.touch_id;
                    return Some(i);
                }
            }
        }
        None   // 非 Down 无主 / 触摸槽满 → 丢弃

    /// 消费本帧输入 → 产 EventRecord 序列。
    pub fn process(&mut self, scene: &mut Scene, events: &[PointerEvent]) -> Vec<EventRecord> {
        let mut out: Vec<EventRecord> = Vec::new();
        if events.is_empty() {
            for i in 0..self.slots.len() {
                if i == 0 || self.slots[i].touch_id >= 0 {   // 活跃槽
                    Self::hover_diff_slot(&mut self.slots[i], scene, &mut out);
                }
            }
            self.recompute_hovered(scene);
            self.recompute_active(scene);
            return out;
        }
        for ev in events {
            let slot_idx = match self.find_or_alloc_slot(ev) { Some(i) => i, None => continue };
            let slot = &mut self.slots[slot_idx];
            slot.last_pos = (ev.x, ev.y);
            let hit = hit_test(scene, slot.last_pos);
            slot.last_hit = hit;
            let touch_id = ev.touch_id;
            match ev.kind {
                PointerKind::Move => {
                    Self::hover_diff_slot(slot, scene, &mut out);
                    // Move 派发：有 monitor 产 Move@monitor（T2 实现），无 monitor 不产
                    for m in &slot.touch_monitors {
                        out.push(EventRecord { node_id: m.0 as u32, event_type: EVT_MOVE, touch_id, x: ev.x, y: ev.y });
                    }
                }
                PointerKind::Down => {
                    slot.is_down = true;
                    slot.down_pos = (ev.x, ev.y);
                    slot.down_node = hit;
                    if let Some(n) = hit {
                        if !scene.nodes[n.0].disabled {
                            out.push(EventRecord { node_id: n.0 as u32, event_type: EVT_DOWN, touch_id, x: ev.x, y: ev.y });
                        }
                    }
                    Self::hover_diff_slot(slot, scene, &mut out);
                }
                PointerKind::Up => {
                    slot.is_down = false;
                    if let Some(n) = hit {
                        if !scene.nodes[n.0].disabled {
                            out.push(EventRecord { node_id: n.0 as u32, event_type: EVT_UP, touch_id, x: ev.x, y: ev.y });
                            if self.slot_click_ok(slot_idx, n) {

                            }
                        }

                            out.push(EventRecord { node_id: m.0 as u32, event_type: EVT_UP, touch_id, x: ev.x, y: ev.y });
                        }
                    }
                    slot.touch_monitors.clear();
                    slot.down_node = None;
                    Self::hover_diff_slot(slot, scene, &mut out);
                    if slot_idx > 0 { slot.touch_id = -1; }   // 释放触摸槽（鼠标不释放）
                }
            }
        }
        self.recompute_hovered(scene);
        self.recompute_active(scene);
        out
    }

    /// click 判定（沿用 v1c.2：down_node==hit && 位移<10px）。slot_idx 取该槽 down_node/down_pos。
    fn slot_click_ok(&self, slot_idx: usize, hit: NodeId) -> bool {
        let slot = &self.slots[slot_idx];
        if slot.down_node != Some(hit) { return false; }
        let dx = slot.last_pos.0 - slot.down_pos.0;
        let dy = slot.last_pos.1 - slot.down_pos.1;
        (dx * dx + dy * dy).sqrt() <= CLICK_THRESHOLD_PX
    }

    /// per-slot hover diff：产 RollOut(旧链独有)/RollOver(新链独有)。不调 set_hovered_chain（全局 union 在 recompute_hovered）。
    fn hover_diff_slot(slot: &mut TouchSlot, scene: &mut Scene, out: &mut Vec<EventRecord>) {
        let new_chain = ancestor_chain(scene, slot.last_hit);
        if new_chain == slot.last_hovered_chain { return; }
        for n in &slot.last_hovered_chain {
            if !new_chain.contains(n) {
                out.push(EventRecord { node_id: n.0 as u32, event_type: EVT_ROLL_OUT, touch_id: slot.touch_id, x: slot.last_pos.0, y: slot.last_pos.1 });
            }
        }
        for n in &new_chain {
            if !slot.last_hovered_chain.contains(n) {
                out.push(EventRecord { node_id: n.0 as u32, event_type: EVT_ROLL_OVER, touch_id: slot.touch_id, x: slot.last_pos.0, y: slot.last_pos.1 });
            }
        }
        slot.last_hovered_chain = new_chain;
    }

    /// 全局 hovered 合并：清所有 → 所有活跃槽命中链 union（任一指命中元素或祖先 → :hover）。
    fn recompute_hovered(&self, scene: &mut Scene) {
        for n in scene.nodes.iter_mut() { n.hovered = false; }
        for i in 0..self.slots.len() {
            if i == 0 || self.slots[i].touch_id >= 0 {
                let mut cur = self.slots[i].last_hit;
                while let Some(id) = cur {
                    if id.0 >= scene.nodes.len() { break; }
                    scene.nodes[id.0].hovered = true;
                    cur = scene.nodes[id.0].parent;
                }
            }
        }
    }

    /// 全局 active 合并：清所有 → 所有 is_down 槽的 down_node 命中链 union（基于 down_node，Down 时命中）。
    fn recompute_active(&self, scene: &mut Scene) {
        for n in scene.nodes.iter_mut() { n.active = false; }
        for slot in &self.slots {
            if slot.is_down {
                let mut cur = slot.down_node;
                while let Some(id) = cur {
                    if id.0 >= scene.nodes.len() { break; }
                    scene.nodes[id.0].active = true;
                    cur = scene.nodes[id.0].parent;
                }
            }
        }
    }
}

    /// 加 touch monitor（去重）。touch_id 找槽（鼠标=-1→slot0）；找不到槽→no-op（Down 前调无效）。
    /// 照 fgui AddTouchMonitor：仅加指定槽，不做 -1 广播（fgui 自身不用）。
    pub fn add_touch_monitor(&mut self, touch_id: i32, node: NodeId) {
        let slot_idx = if touch_id == -1 { 0 } else {
            match (1..self.slots.len()).find(|&i| self.slots[i].touch_id == touch_id) { Some(i) => i, None => return }
        };
        let slot = &mut self.slots[slot_idx];
        if !slot.touch_monitors.contains(&node) {
            slot.touch_monitors.push(node);
        }
    }

    /// 移除 touch monitor（从所有槽）。照 fgui RemoveTouchMonitor：置 sentinel 而非 RemoveAt（避免遍历偏移）。
    pub fn remove_touch_monitor(&mut self, node: NodeId) {
        for slot in &mut self.slots {
            // touch_monitors 是 Vec<NodeId>，用 retain 移除（Vec 无 sentinel 需求，retain 更简且无遍历期偏移）
            slot.touch_monitors.retain(|n| *n != node);
        }
    }
```

> 注：fgui 用 `touchMonitors[i] = null` + 遍历时跳 null，因 C# List 遍历中 RemoveAt 偏移。Rust Vec 用 `retain` 一次性移除更干净（不在派发遍历中调，派发在 process 内一次性消费后 clear）。retain 等价且更安全。

        void BubbleRoute(LoomEvent evt)
        {
            var chain = AncestorChain(evt.nodeId);
            var ctx = EventContext.Get();
            ctx.target = evt.nodeId; ctx.type = evt.type; ctx.touchId = evt.touch_id; ctx.x = evt.x; ctx.y = evt.y;
            // capture 阶段：根→target 反向，全跑（照 fgui 不检查 stop）。消费 _touchCapture 记 _captureNodeCap。
            for (int i = chain.Count - 1; i >= 0; i--)
            {
                ctx.currentTarget = chain[i]; ctx.phase = Phase.Capture;
                TryGetBridge(chain[i], evt.type)?.CallCapture(ctx);
                if (ctx._touchCapture) { ctx._touchCapture = false; _captureNodeCap = chain[i]; }
            }
            // bubble 阶段：target→root 正向，stop break。消费 _touchCapture 记 _captureNodeBub。
            if (!ctx._stopsPropagation)
                for (int i = 0; i < chain.Count; i++)
                {
                    ctx.currentTarget = chain[i];
                    ctx.phase = (chain[i] == ctx.target) ? Phase.Target : Phase.Bubble;
                    TryGetBridge(chain[i], evt.type)?.CallBubble(ctx);
                    if (ctx._touchCapture) { ctx._touchCapture = false; _captureNodeBub = chain[i]; }
                    if (ctx._stopsPropagation) break;
                }
            EventContext.Return(ctx);
        }
```

        public void DispatchPending(IntPtr ptr, int count)
        {
            if (ptr == IntPtr.Zero || count <= 0) return;
            int recSize = Marshal.SizeOf<LoomEvent>();
            for (int i = 0; i < count; i++)
            {
                var evt = Marshal.PtrToStructure<LoomEvent>(ptr + i * recSize);
                _captureNodeCap = null; _captureNodeBub = null;   // 每事件重置
                switch (evt.type)
                {

                        BubbleRoute(evt);
                        // Down 路由后，capture 阶段 + bubble 阶段各消费一次 → 各加一 monitor（照 fgui）

                        if (_captureNodeBub.HasValue)

                        break;
                    case EventType.Up:
                    case EventType.Click:
                        BubbleRoute(evt); break;
                    case EventType.Move:
                        DirectDispatch(evt); break;   // v1c.3：Move 改直派（核心算好的 monitor 目标）
                    case EventType.RollOver:
                    case EventType.RollOut:
                        DirectDispatch(evt); break;
                }
            }
        }
