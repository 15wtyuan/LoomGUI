# LoomGUI v0 骨架 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 纯 Rust 端到端骨架：HTML/CSS fixture → parse → style → scene → text → layout → render → `stage.tick` → `render_nodes` JSON 快照，验证 Rust 核心能力 + 解耦 FFI/Unity 风险。

**Architecture:** 单 workspace，`loomgui_core` 一个 lib crate（砍 `event`/`anim` 两模块——静态快照不接输入不跑动画）+ `examples/v0_snapshot.rs`（runner）+ `tests/snapshot.rs`（insta 快照）。内存直通（parse→scene 直喂，打包器 defer v1）；文本测量全做（ttf-parser + unicode-linebreak → TextLayout SOA 三表）；纹理占位 `tex_id`（不加载 PNG）。每帧 `tick(空输入, dt=0)` 首帧直出 `Vec<RenderNode>`，serde 序列化为 JSON。

**Tech Stack:** Rust（edition 2021）、`scraper`（HTML）、`cssparser`（CSS 声明）、`taffy`（flexbox）、`ttf-parser`（字体度量）、`unicode-linebreak`（断行）、`serde`/`serde_json`（序列化）、`insta`（快照测试）。

**依据:** spec `docs/superpowers/specs/2026-06-18-v0-skeleton-design.md`、主设计 `docs/design/00-main-design.md`（§5 parse/style、§6 Node、§7 taffy、§8 render、§9 text）、围栏 `docs/roadmap/v1-scope.md` §2。

## Global Constraints

- **Rust edition 2021**，workspace 单 crate `loomgui_core`。
- **围栏元素**（v1-scope §2）：`div`(Container) / `span`+裸文本(Text) / `img`(Image) / `button`(Button)。围栏外元素解析报错，不降级。
- **CSS 子集**（v1-scope §2）：见各属性 mapping（Task 3）。围栏外/非法值取 `taffy::Style::DEFAULT` + 告警；布局语义违规（display 非 flex/none、position:absolute/sticky/fixed、float、grid、行内混排）编译期报错。
- **行内混排报错**：元素内「文本+元素+文本」混排、多 span 拼行内、`<b>/<i>/<u>` 在普通 div/span 内——解析期报错，提示用单个文本或 `<l-rich>`。不降级。
- **坐标系**（§8.1）：核心统一左上原点、y 向下，核心代码不出现 `height-y` 翻转。
- **内存直通**：v0 parse→scene 直喂，跳过打包器磁盘包（defer v1 第一阶段）。
- **文本全做**：ttf-parser 度量 + unicode-linebreak 断行 → TextLayout SOA 三表，砍 rustybuzz/BiDi/fallback。core 文本模块对任意 unicode 透明（按字形有无给 glyph_id，缺字形给 0）；**v0 测试 fixture 锁 ASCII 文本**（DejaVu Sans 无 CJK glyph，见下"字体"项），CJK 文本渲染验证留 v1。
- **纹理占位**：img 的 src 不加载，→ 占位 `tex_id`（src 哈希或固定 id），UV 占位全图。纹理加载是后端职责，v0 不碰。
- **serde derive**：`RenderNode`/`NodePayload`/`NodeTransform`/`MaskContext`/`BlendMode` 派生 `Serialize`，`serde_json::to_string_pretty` 产 JSON。
- **静态首帧**：`Stage::tick` 接空输入、`dt=0`，首帧直出 `Vec<RenderNode>`。v0 无 event/anim/ScrollPane。
- **字体**：v0 单字体无 fallback，从文件路径加载一个 ttf。**测试字体锁仓库内 `loomgui_core/tests/fixtures/DejaVuSans.ttf`**（开源 DejaVu 字体，许可见同目录 `FONT_LICENSE.txt`，允许再分发；~740 KiB）——跨平台一致，不依赖系统 arial.ttf（Linux CI 无 arial 会漂移）。测试用 `env!("CARGO_MANIFEST_DIR")` 拼绝对路径指向该 fixture，`skip_if_no_font` 作防御性保留（正常仓库内字体必存在，实际不会 skip）。DejaVu Sans 无 CJK glyph，故 v0 fixture 文本用 ASCII；CJK 文本渲染验证留 v1（需 CJK 字体策略，见 §9）。

```rust
pub fn parse_html(html: &str) -> Result<ElementTree, String> {
    let document = Html::parse_document(html);
    // 取根（body 子节点，忽略 html/head/body 包裹）
    let body_sel = ScraperSelector::parse("body").unwrap();
    let body = document.select(&body_sel).next().ok_or("no body")?;

    let mut tree = ElementTree { roots: Vec::new(), nodes: Vec::new() };
    for child_node in body.children() {
        if let Some(child_el) = child_node.value().as_element() {
            let id = build_element(&mut tree, &document, child_el, child_node, None)?;
            tree.roots.push(id);
        }
    }
    Ok(tree)
}

fn build_element(
    tree: &mut ElementTree,
    doc: &Html,
    el_val: &scraper::node::Element,
    el_node: scraper::ElementRef,
    parent: Option<ElementId>,
) -> Result<ElementId, String> {
    let tag = el_val.name().to_string();
    let id_attr = el_val.attr("id").map(|s| s.to_string());
    let classes = el_val
        .attr("class")
        .map(|s| s.split_whitespace().map(|w| w.to_string()).collect())
        .unwrap_or_default();

    // 收集直接文本子节点；若同时有文本子节点和元素子节点 → 行内混排报错
    let mut text_parts: Vec<String> = Vec::new();
    let mut element_children: Vec<scraper::ElementRef> = Vec::new();
    for child in el_node.children() {
        match child.value() {
            scraper::node::Node::Text(t) => {
                let s = t.text.trim();
                if !s.is_empty() {
                    text_parts.push(s.to_string());
                }
            }
            scraper::node::Node::Element(_) => {
                if let Some(eref) = scraper::ElementRef::wrap(child) {
                    element_children.push(eref);
                }
            }
            _ => {}
        }
    }

    let has_text = !text_parts.is_empty();
    let has_elements = !element_children.is_empty();
    // 叶子文本元素（span/裸文本/button 文本）允许 text 无 element 子；
    // 容器（div 等）允许 element 子无 text；混排报错。
    if has_text && has_elements {
        return Err(format!(
            "行内混排不支持（div 只装 flex item）：元素 <{}> 同时含文本与子元素，改用单个文本或 l-rich",
            tag
        ));
    }

    let text = if has_text {
        Some(text_parts.join(" "))
    } else {
        None
    };

    let idx = tree.nodes.len();
    tree.nodes.push(ElementData {
        tag,
        classes,
        id: id_attr,
        text,
        children: Vec::new(),
        parent,
    });
    let my_id = ElementId(idx);

    let mut children_ids = Vec::new();
    for child_el in element_children {
        let child_val = child_el.value();
        let cid = build_element(tree, doc, child_val, child_el, Some(my_id))?;
        children_ids.push(cid);
    }
    tree.nodes[idx].children = children_ids;
    Ok(my_id)
}
```

cssparser 0.34 的 API 较底层，用最直接的 `StyleSheetParser`：

```rust
use cssparser::{
    parse_one_rule, AtRuleParser, BasicParseError, DeclarationParser, ParseError,
    Parser, ParserInput, QualifiedRuleParser, RuleBodyItemParser, RuleBodyParser,
    ToCss, Token,
};

struct NestingParser;

impl<'i> QualifiedRuleParser<'i> for NestingParser {
    type Prelude = String;       // 选择器文本
    type Error = ();

    fn parse_prelude<'t>(
        &mut self,
        input: &mut Parser<'i, 't>,
    ) -> Result<Self::Prelude, ParseError<'i, Self::Error>> {
        let start = input.position();
        while input.next().is_ok() {}
        Ok(input.slice_from(start).trim().to_string())
    }

    fn parse_block<'t>(
        &mut self,
        prelude: Self::Prelude,
        _start: &cssparser::SourcePosition,
        input: &mut Parser<'i, 't>,
    ) -> Result<Rule, ParseError<'i, Self::Error>> {
        let mut declarations = Vec::new();
        let parser = &mut RuleBodyParser::new(input, &mut DeclParser);
        while let Some(item) = parser.next() {
            if let Ok(d) = item {
                declarations.push(d);
            }
        }
        Ok(Rule { selector_text: prelude, declarations })
    }
}

impl<'i> AtRuleParser<'i> for NestingParser {
    type Prelude = ();
    type AtRule = Rule;
    type Error = ();
    // v0 无 @ 规则，全部 parse_block 默认拒绝
}

struct DeclParser;
impl<'i> RuleBodyItemParser<'i, Declaration, ()> for DeclParser {
    fn parse_qualified(&mut self) -> bool { true }
    fn parse_declarations(&mut self) -> bool { true }
}
impl<'i> DeclarationParser<'i> for DeclParser {
    type Declaration = Declaration;
    type Error = ();
    fn parse_value<'t>(
        &mut self,
        name: cssparser::CowRcStr<'i>,
        input: &mut Parser<'i, 't>,
    ) -> Result<Declaration, ParseError<'i, ()>> {
        let start = input.position();
        while input.next().is_ok() {}
        let value = input.slice_from(start).trim().to_string();
        Ok(Declaration { prop: name.to_string(), value })
    }
}

pub fn parse_css(css: &str) -> Result<StyleSheet, String> {
    let mut input = ParserInput::new(css);
    let mut parser = Parser::new(&mut input);
    let mut rules = Vec::new();
    let rule_parser = &mut NestingParser;
    loop {
        match parse_one_rule(&mut parser, rule_parser) {
            Ok(rule) => rules.push(rule),
            Err(_) => break,
        }
    }
    Ok(StyleSheet { rules })
}
```

> 注：cssparser 0.34 的精确 API 可能在编译时有微调。若 `parse_one_rule`/`RuleBodyParser` 签名不符，按编译器提示调整——核心是「遍历规则、每规则取选择器文本 + 声明列表」。这是实现期唯一允许对文档代码做适配的地方。

```rust
#[derive(Debug, Clone)]
pub struct ParsedSelector {
    pub raw: String,
    pub compound: Vec<Compound>,
    pub specificity: Specificity,
}

#[derive(Debug, Clone)]
pub struct Compound {
    pub tag: Option<String>,
    pub classes: Vec<String>,
    pub id: Option<String>,
    pub combinator: Combinator,
}

#[derive(Debug, Clone, Copy, PartialEq)]
pub enum Combinator { Descendant, Child }

#[derive(Debug, Clone, Copy, PartialEq, Eq, PartialOrd, Ord)]
pub struct Specificity(pub u32, pub u32, pub u32);

/// 极简解析：按空格切 descendant，`>` 切 child；复合内 tag/.class/#id。
pub fn parse_selector(raw: &str) -> Result<ParsedSelector, String> {
    let raw = raw.trim().to_string();
    // 先按子代 `>` 拆，再按空格拆后代。v0 不支持 `+`/`~`。
    let mut compound: Vec<Compound> = Vec::new();
    let mut combinator = Combinator::Descendant; // 第一个 compound 的关系无意义
    let mut tokens: Vec<(String, bool)> = Vec::new(); // (text, is_child_boundary)
    let mut cur = String::new();
    let mut chars = raw.chars().peekable();
    let mut first = true;
    // 简单 tokenizer：遇 `>` 记 child，遇空格记 descendant（折叠多余空白）
    let mut parts: Vec<(String, Combinator)> = Vec::new();
    let mut buf = String::new();
    let mut next_comb = Combinator::Descendant;
    let mut just_seen_space = false;
    for ch in raw.chars() {
        match ch {
            '>' => {
                if !buf.trim().is_empty() { parts.push((buf.trim().to_string(), if first {Combinator::Descendant} else {next_comb})); first=false; }
                buf.clear();
                next_comb = Combinator::Child;
                just_seen_space = false;
            }
            c if c.is_whitespace() => {
                if !buf.trim().is_empty() { parts.push((buf.trim().to_string(), if first {Combinator::Descendant} else {next_comb})); first=false; buf.clear(); }
                next_comb = Combinator::Descendant;
                just_seen_space = true;
            }
            c => { buf.push(c); just_seen_space = false; }
        }
    }
    if !buf.trim().is_empty() {
        parts.push((buf.trim().to_string(), if first {Combinator::Descendant} else {next_comb}));
    }

    let mut spec = Specificity(0,0,0);
    for (text, comb) in &parts {
        let mut tag = None;
        let mut classes = Vec::new();
        let mut id = None;
        let mut t = String::new();
        for ch in text.chars() {
            match ch {
                '.' => { if !t.is_empty() { tag = Some(std::mem::take(&mut t)); } }
                '#' => { if !t.is_empty() { tag = Some(std::mem::take(&mut t)); } t.clear(); /* mark id follow */ }
                _ => {}
            }
            t.push(ch);
        }
        // 重新解析（上面逻辑简化重写）
        tag = None; classes.clear(); id = None;
        let bytes = text.as_bytes();
        let mut i = 0;
        let mut cur_token = String::new();
        let mut token_kind = 't'; // t=tag, .=class, #=id
        let mut push_token = |kind: char, val: &str, tag: &mut Option<String>, classes: &mut Vec<String>, id: &mut Option<String>| {
            if val.is_empty() { return; }
            match kind {
                't' => *tag = Some(val.to_string()),
                '.' => classes.push(val.to_string()),
                '#' => *id = Some(val.to_string()),
                _ => {}
            }
        };
        let mut idx = 0;
        let mut kind = 't';
        let mut cur = String::new();
        while idx < bytes.len() {
            let c = bytes[idx] as char;
            if c == '.' || c == '#' {
                push_token(kind, &cur, &mut tag, &mut classes, &mut id);
                cur.clear();
                kind = c;
            } else {
                cur.push(c);
            }
            idx += 1;
        }
        push_token(kind, &cur, &mut tag, &mut classes, &mut id);

        if id.is_some() { spec.0 += 1; }
        spec.1 += classes.len() as u32;
        if tag.is_some() { spec.2 += 1; }
        compound.push(Compound { tag, classes, id, combinator: *comb });
    }

    Ok(ParsedSelector { raw, compound, specificity: spec })
}

fn compound_matches(c: &Compound, el: &ElementData) -> bool {
    if let Some(t) = &c.tag { if el.tag != *t { return false; } }
    if let Some(id) = &c.id { if el.id.as_deref() != Some(id.as_str()) { return false; } }
    for cls in &c.classes {
        if !el.classes.iter().any(|c| c == cls) { return false; }
    }
    true
}

/// 从右往左匹配：最后一个 compound 必须命中目标元素，前面的按 combinator 沿父链找。
fn matches(sel: &ParsedSelector, el_id: ElementId, tree: &ElementTree) -> bool {
    let comps = &sel.compound;
    if comps.is_empty() { return false; }
    let last = &comps[comps.len()-1];
    if !compound_matches(last, &tree.nodes[el_id.0]) { return false; }
    if comps.len() == 1 { return true; }
    // 向上匹配剩余 compound（从倒数第二个到第一个）
    let mut cur_el = el_id;
    let mut idx = comps.len() - 1;
    while idx > 0 {
        let target_comp = &comps[idx - 1];
        let parent = tree.nodes[cur_el.0].parent;
        let parent = match parent { Some(p) => p, None => return false };
        if !compound_matches(target_comp, &tree.nodes[parent.0]) { return false; }
        cur_el = parent;
        idx -= 1;
        // v0 简化：不严格区分 descendant/child 的多级，只查直接父。
        // （child 要求直接父；descendant 也只查父——v0 树浅，够用；如需严格后代遍历，实现期补）
    }
    true
}

pub fn match_element<'a>(
    el: &ElementData,
    tree: &'a ElementTree,
    rules: &'a [Rule],
) -> Vec<&'a Rule> {
    // 找目标元素 id
    let el_id = tree.nodes.iter().position(|n| std::ptr::eq(n, el)).map(ElementId);
    let el_id = match el_id { Some(id) => id, None => return Vec::new() };

    let mut matched: Vec<(Specificity, usize, &Rule)> = Vec::new();
    for (i, rule) in rules.iter().enumerate() {
        let sel = match parse_selector(&rule.selector_text) { Ok(s) => s, Err(_) => continue };
        if matches(&sel, el_id, tree) {
            matched.push((sel.specificity, i, rule));
        }
    }
    // specificity 降序，同级按出现顺序（i 升序）
    matched.sort_by(|a, b| b.0.cmp(&a.0).then(a.1.cmp(&b.1)));
    matched.into_iter().map(|(_, _, r)| r).collect()
}
```

> 注：上面 `parse_selector` 内有两段重复的 token 解析逻辑（第一段未用，第二段实际生效）——这是简化草稿。实现期清理掉第一段死代码，保留 `idx` 那段循环即可。功能以测试为准。

```rust
/// px → Dimension::Length(f32)；% → LengthPercentage::Percent；auto → Auto
pub fn parse_length(s: &str) -> LengthPercentageAuto {
    let s = s.trim();
    if s == "auto" { return LengthPercentageAuto::Auto; }
    parse_lp(s).into()
}

pub fn parse_lp(s: &str) -> LengthPercentage {
    let s = s.trim();
    if let Some(pct) = s.strip_suffix('%') {
        if let Ok(v) = pct.trim().parse::<f32>() {
            return LengthPercentage::Percent(v / 100.0);
        }
    }
    if let Some(px) = s.strip_suffix("px") {
        if let Ok(v) = px.trim().parse::<f32>() { return LengthPercentage::Length(v); }
    }
    // 裸数字当 px
    if let Ok(v) = s.parse::<f32>() { return LengthPercentage::Length(v); }
    LengthPercentage::Length(0.0)
}

pub fn parse_dimension(s: &str) -> Dimension {
    match parse_lp(s) {
        LengthPercentage::Length(v) => Dimension::Length(v),
        LengthPercentage::Percent(v) => Dimension::Percent(v),
    }
}

/// 1~4 值展开四向（top right bottom left）
pub fn parse_four(s: &str) -> [f32; 4] {
    let parts: Vec<&str> = s.split_whitespace().collect();
    let p = |i: usize| -> f32 {
        parts.get(i).and_then(|x| x.strip_suffix("px").unwrap_or(x).trim().parse::<f32>().ok()).unwrap_or(0.0)
    };
    match parts.len() {
        1 => { let v = p(0); [v, v, v, v] }
        2 => { let v = p(0); let h = p(1); [v, h, v, h] }
        3 => [p(0), p(1), p(2), p(1)],
        _ => [p(0), p(1), p(2), p(3)],
    }
}

pub fn parse_color(s: &str) -> Option<[f32; 4]> {
    let s = s.trim();
    let s = s.strip_prefix('#').unwrap_or(s);
    if s.len() == 6 {
        let r = u8::from_str_radix(&s[0..2], 16).ok()?;
        let g = u8::from_str_radix(&s[2..4], 16).ok()?;
        let b = u8::from_str_radix(&s[4..6], 16).ok()?;
        Some([r as f32/255.0, g as f32/255.0, b as f32/255.0, 1.0])
    } else { None }
}
```

```rust
/// 给整棵树算每元素的 ResolvedStyle（继承在 resolve 期展开）。
/// 索引与 ElementTree.nodes 一一对应。
pub fn resolve_styles(tree: &ElementTree, sheet: &StyleSheet) -> Vec<ResolvedStyle> {
    let mut out: Vec<ResolvedStyle> = (0..tree.nodes.len()).map(|_| ResolvedStyle::default()).collect();
    // 根元素用默认；自顶向下，每元素从父继承白名单后再叠自己的命中规则。
    fn resolve_rec(
        tree: &ElementTree, sheet: &StyleSheet, id: ElementId,
        parent: Option<&ResolvedStyle>, out: &mut [ResolvedStyle],
    ) {
        let mut style = ResolvedStyle::default();
        if let Some(p) = parent {
            // 继承白名单字段
            style.color = p.color;
            style.font_size = p.font_size;
            style.font_family = p.font_family.clone();
            style.font_weight = p.font_weight;
            style.line_height = p.line_height;
            style.letter_spacing = p.letter_spacing;
            style.text_align = p.text_align;
            style.white_space_nowrap = p.white_space_nowrap;
        }
        // img src 从属性拿（v0：parse 时没存 attr，这里从 text 或约定拿；实际在 scene 层从 ElementData 扩展取）
        // —— 见 scene task，这里只管 CSS。

        let el = &tree.nodes[id.0];
        let rules = match_element(el, tree, &sheet.rules);
        for rule in rules {
            for decl in &rule.declarations {
                apply_decl(&mut style, &decl.prop, &decl.value);
            }
        }
        out[id.0] = style;
        for child in &el.children {
            resolve_rec(tree, sheet, *child, Some(&style), out);
        }
    }
    for root in &tree.roots {
        resolve_rec(tree, sheet, *root, None, &mut out);
    }
    out
}
```

```rust
pub fn build_scene(tree: &ElementTree, styles: &[ResolvedStyle]) -> Scene {
    let mut scene = Scene { roots: Vec::new(), nodes: Vec::new() };
    for root in &tree.roots {
        let id = build_rec(tree, styles, *root, None, &mut scene);

fn build_rec(
    tree: &ElementTree, styles: &[ResolvedStyle], el_id: ElementId,
    parent: Option<NodeId>, scene: &mut Scene,
) -> NodeId {
    let el = &tree.nodes[el_id.0];
    let style = &styles[el_id.0];
    let kind = match el.tag.as_str() {
        "div" | "l-container" => NodeKind::Container,
        "button" => NodeKind::Button,
        "img" => NodeKind::Image { src: el.text.clone().unwrap_or_default() }, // v0: src 暂存 text；见下注

    };
    let has_children = !el.children.is_empty();
    let nid = NodeId(scene.nodes.len());
    scene.nodes.push(Node {
        id: nid, parent, kind, style: style.clone(),
        taffy_id: None, layout_rect: Rect::default(),
        clip_rect: if style.overflow_hidden { Some(Rect::default()) } else { None },
        children: Vec::new(),
        dirty_mesh: true, dirty_text: matches!(kind, NodeKind::Text { .. }),
    });
    if has_children {
        let mut kids = Vec::new();
        for c in &el.children {
            let cid = build_rec(tree, styles, *c, Some(nid), scene);
            kids.push(cid);
        }
        scene.nodes[nid.0].children = kids;
    }
    nid
}
```

```rust
pub fn measure_text(
    content: &str,
    font_size: f32,
    line_height: f32,
    letter_spacing: f32,
    align: crate::style::resolved::TextAlign,
    nowrap: bool,
    max_width: Option<f32>,
    font: &Font,
) -> TextLayout {
    let ascent = font.ascent(font_size);
    let descent = font.descent(font_size);   // 负
    let line_gap = font.line_gap(font_size);
    // Line.height：line_height 倍数生效则 font_size*倍数；否则 ascent - descent + line_gap
    let line_h = if line_height > 0.0 { font_size * line_height } else { ascent - descent + line_gap };
    let baseline = if line_height > 0.0 { (line_h + ascent - descent) / 2.0 - (descent).abs() } else { ascent };
    // ↑ baseline 简化：v0 对齐 Chrome 留作实现期调（主文档 §9.1 原则）

    // 断行：unicode-linebreak 给换行机会点。v0 简化——按空白 + 宽度约束贪心切。
    let words: Vec<&str> = content.split(' ').collect();
    let mut lines: Vec<(String, f32)> = Vec::new(); // (text, width)
    let measure_width = |s: &str| -> f32 {
        let mut pen = 0.0f32;
        let mut prev: Option<u16> = None;
        for ch in s.chars() {
            let gid = font.face.glyph_index(ch).map(|g| g.0).unwrap_or(0);
            if let Some(p) = prev {
                if let Some(k) = font.face.kerning_for(prev.unwrap_or(0), gid) { pen += k as f32 / font.face.units_per_em() as f32 * font_size; }
            }
            let adv = font.face.glyph_advance_width(gid).unwrap_or(0) as f32 / font.face.units_per_em() as f32 * font_size;
            pen += adv + letter_spacing;
            prev = Some(gid);
        }
        pen
    };

    let max_w = max_width.unwrap_or(f32::MAX);
    let mut cur = String::new();
    let mut cur_w = 0.0f32;
    for word in &words {
        let ww = measure_width(word);
        let sep = if cur.is_empty() { 0.0 } else { measure_width(" ") };
        if nowrap || cur.is_empty() || cur_w + sep + ww <= max_w {
            if !cur.is_empty() { cur.push(' '); cur_w += sep; }
            cur.push_str(word); cur_w += ww;
        } else {
            lines.push((std::mem::take(&mut cur), cur_w));
            cur.push_str(word); cur_w = ww;
        }
    }
    if !cur.is_empty() { lines.push((cur, cur_w)); }
    if lines.is_empty() { lines.push((String::new(), 0.0)); }

    let text_width = lines.iter().map(|(_, w)| *w).fold(0.0f32, f32::max);
    let text_height = lines.len() as f32 * line_h;

    // 生成 glyphs（绝对坐标，§9.2：已累加 advance + 已应用 align 偏移）
    let mut out_lines = Vec::new();
    for (li, (text, lw)) in lines.iter().enumerate() {
        let line_y = li as f32 * line_h;
        let x_offset = match align {
            crate::style::resolved::TextAlign::Center => (text_width - lw) / 2.0,
            crate::style::resolved::TextAlign::Right => text_width - lw,
            _ => 0.0,
        };
        let mut pen_x = x_offset;
        let mut glyphs = Vec::new();
        let mut prev: Option<u16> = None;
        for ch in text.chars() {
            let gid = font.face.glyph_index(ch).map(|g| g.0).unwrap_or(0);
            if let Some(p) = prev {
                if let Some(k) = font.face.kerning_for(p, gid) { pen_x += k as f32 / font.face.units_per_em() as f32 * font_size; }
            }
            // bearing from glyph bbox
            let (bx, by) = font.face.glyph_bounding_box(gid).map(|b| {
                (b.x_min as f32 / font.face.units_per_em() as f32 * font_size,
                 b.y_max as f32 / font.face.units_per_em() as f32 * font_size)
            }).unwrap_or((0.0, 0.0));
            glyphs.push(Glyph { glyph_id: gid, x: pen_x, y: line_y, bearing_x: bx, bearing_y: by });
            let adv = font.face.glyph_advance_width(gid).unwrap_or(0) as f32 / font.face.units_per_em() as f32 * font_size;
            pen_x += adv + letter_spacing;
            prev = Some(gid);
        }
        out_lines.push(Line {
            y: line_y, height: line_h, baseline: line_y + baseline, width: *lw,
            runs: vec![GlyphRun { font_size, glyphs }],
        });
    }

    TextLayout { text_width, text_height, lines: out_lines }
}
```

```rust
/// 就地 solve：建 taffy 树 → 注册 MeasureFunc → compute_layout → 回写 layout_rect。
pub fn solve(scene: &mut Scene, font: &Font, root_size: (f32, f32)) {
    let mut taffy_tree: TaffyTree<()> = TaffyTree::new();
    // 1. 为每个 Node 建 taffy 节点，存映射
    let mut taffy_ids: Vec<Option<NodeId>> = vec![None; scene.nodes.len()];
    let mut map: Vec<taffy::NodeId> = Vec::with_capacity(scene.nodes.len());

    fn build(scene: &Scene, font: &Font, tree: &mut TaffyTree<()>, taffy_ids: &mut Vec<Option<taffy::NodeId>>, id: NodeId) -> taffy::NodeId {
        let node = &scene.nodes[id.0];
        let mut style = node.style.taffy_style.clone();
        let mut ctx = ();
        // 叶子：Text/Image 用 MeasureFunc
        let measured = match &node.kind {
            NodeKind::Text { content } => {
                let s = &node.style;
                let content = content.clone();
                let font_size = s.font_size;
                let lh = s.line_height;
                let ls = s.letter_spacing;
                let align = s.text_align;
                let nowrap = s.white_space_nowrap;
                let measure = move |known: Size<Option<f32>>, _avail: Size<AvailableSpace>, _ctx| -> Size<f32> {
                    let max_w = known.width; // Option<f32>：Some=约束, None=不限
                    let layout = measure_text(&content, font_size, lh, ls, align, nowrap, max_w, font);
                    Size { width: layout.text_width, height: layout.text_height }
                };
                Some(MeasureFunc::Boxed(Box::new(measure)))
            }
            NodeKind::Image { src: _ } => {
                // v0 占位：图片 intrinsic 用声明尺寸或 64x64 占位
                let s = &node.style.taffy_style;
                let w = if let Dimension::Length(v) = s.size.width { v } else { 64.0 };
                let h = if let Dimension::Length(v) = s.size.height { v } else { 64.0 };
                let measure = move |_known, _avail, _ctx| Size { width: w, height: h };
                Some(MeasureFunc::Boxed(Box::new(measure)))
            }
            _ => None,
        };

        let children_ids: Vec<taffy::NodeId> = node.children.iter().map(|c| build(scene, font, tree, taffy_ids, *c)).collect();
        let tid = if let Some(m) = measured {
            tree.new_leaf_with_context(style, m).unwrap()
        } else {
            tree.new_with_children(style, children_ids).unwrap()
        };
        taffy_ids[id.0] = Some(tid);
        tid
    }

    let root_tid = build(scene, font, &mut taffy_tree, &mut map, scene.roots[0]);
    // 设根 size
    taffy_tree.set_style(root_tid, Style {
        size: Size { width: LengthPercentageAuto::Length(root_size.0), height: LengthPercentageAuto::Length(root_size.1) },
        ..taffy_tree.style(root_tid).unwrap().clone()
    }).ok();

    taffy_tree.compute_layout(root_tid, Size::MAX).ok();

    // 回写 layout_rect + clip_rect
    fn write_back(scene: &mut Scene, tree: &TaffyTree<()>, taffy_ids: &[Option<taffy::NodeId>], id: NodeId, parent_origin: (f32, f32)) {
        let tid = taffy_ids[id.0].unwrap();
        let layout = tree.layout(tid).unwrap();
        let x = parent_origin.0 + layout.location.x;
        let y = parent_origin.1 + layout.location.y;
        let node = &mut scene.nodes[id.0];
        node.layout_rect = Rect { x, y, w: layout.size.width, h: layout.size.height };
        if node.clip_rect.is_some() {
            node.clip_rect = Some(Rect { x, y, w: layout.size.width, h: layout.size.height });
        }
        let kids = node.children.clone();
        for c in kids {
            write_back(scene, tree, taffy_ids, c, (x, y));
        }
    }
    write_back(scene, &taffy_tree, &map, scene.roots[0], (0.0, 0.0));
}
```

```rust
/// v0 简化批合：clip_rect 的 Container 是 BatchingRoot（§8.5）。
/// 绘制序（§8.8）：单一全局计数器，DFS 分配即自增；BatchingRoot 内按树序。
/// sort_key = 绘制序（v0 不做 AABB 重排合并，只保序——重排是 v1.x 优化，v0 保序即正确意图）。
pub fn assign_sort_keys(scene: &Scene, nodes: &mut [RenderNode]) {
    let mut counter: u32 = 0;
    fn dfs(scene: &Scene, nodes: &mut [RenderNode], id: NodeId, counter: &mut u32, parent_mask: MaskContext) {
        let node = &scene.nodes[id.0];
        // mask_context：本节点有 clip_rect → 新层级（v0 简化：层级=出现序）
        let mask = if node.clip_rect.is_some() {
            MaskContext(*counter + 1)  // 简化：用计数器当 id
        } else { parent_mask };
        {
            let rn = &mut nodes[id.0];
            rn.sort_key = *counter;
            rn.mask_context = mask;
            *counter += 1;
        }
        let kids = node.children.clone();
        for c in kids {
            dfs(scene, nodes, c, counter, mask);
        }
    }
    for root in &scene.roots {
        dfs(scene, nodes, *root, &mut counter, MaskContext(0));
    }
}
```

```rust
pub mod mesh;
pub mod batch;

use crate::scene::node::{NodeKind, Scene};
use crate::text::layout::{measure_text, Font};
use node::*;

/// 遍历 Scene → Vec<RenderNode>（顺序与 scene.nodes 索引一致，便于 id 对齐）。
/// Text 节点调 measure_text 产 TextLayout；Container/Image 产 Mesh quad。
pub fn build_render_nodes(scene: &Scene, font: &Font) -> Vec<RenderNode> {
    let mut nodes: Vec<RenderNode> = (0..scene.nodes.len()).map(|_| RenderNode {
        node_id: 0, parent_id: None, visible: true, alpha: 1.0, grayed: false,
        color_tint: [1.0,1.0,1.0,1.0], transform: NodeTransform::default(),
        blend: BlendMode::Normal, mask_context: MaskContext(0), sort_key: 0,
        payload: NodePayload::Unchanged,
    }).collect();

    for n in &scene.nodes {
        let rn = &mut nodes[n.id.0];
        rn.node_id = n.id.0 as u32;
        rn.parent_id = n.parent.map(|p| p.0 as u32);
        rn.alpha = n.style.opacity;
        rn.color_tint = n.style.color;
        rn.transform.x = n.layout_rect.x;
        rn.transform.y = n.layout_rect.y;
        rn.visible = true;
        let rect = &n.layout_rect;
        match &n.kind {
            NodeKind::Container | NodeKind::Button => {
                let color = n.style.background_color.unwrap_or([0.0,0.0,0.0,0.0]);
                let (v,uvc,col,idx) = crate::render::mesh::quad(rect, color);
                rn.payload = NodePayload::Mesh { verts:v, uvs:uvc, colors:col, indices:idx, texture: 0, program: 0 };
            }
            NodeKind::Image { src } => {
                let (v,uvc,col,idx) = crate::render::mesh::quad(rect, [1.0,1.0,1.0,1.0]);
                // 占位 tex_id：src 哈希
                let tex = hash_str(src);
                rn.payload = NodePayload::Mesh { verts:v, uvs:uvc, colors:col, indices:idx, texture: tex, program: 0 };
            }
            NodeKind::Text { content } => {
                let s = &n.style;
                let layout = measure_text(content, s.font_size, s.line_height, s.letter_spacing, s.text_align, s.white_space_nowrap, Some(rect.w), font);
                rn.payload = NodePayload::Text { layout, font_size: s.font_size, color: s.color, program: 1 };
            }
        }
    }
    batch::assign_sort_keys(scene, &mut nodes);
    nodes
}

fn hash_str(s: &str) -> u32 {
    use std::collections::hash_map::DefaultHasher;
    use std::hash::{Hash, Hasher};
    let mut h = DefaultHasher::new();
    s.hash(&mut h);
    (h.finish() & 0xFFFF) as u32
}
```
