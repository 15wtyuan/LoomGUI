# v1b.2 真纹理加载 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让 `<img src="x.png">` 在 Unity 显真像素（非白块），无 CSS 尺寸时按真实像素布局，core 经 TexId 引用、GPU 纹理全在 Unity。

**Architecture:** core 持 per-Stage `TextureRegistry{src→(tex_id,w,h)}`（纯 id+维度表，无 GPU）。Unity load 后 collect srcs→读 PNG→`Texture2D.LoadImage`→`register_texture`（core 分配 tex_id + 存维度）→Unity 建 `tex_id→Texture2D` map。render/measure 查 registry；blob v3 带 `tex_id` 列；MirrorPool 按 tex_id 绑材质（shader/MaterialManager 零改，`tex2D×v.color` 已支持纯色块 + 真图两路）。

**Tech Stack:** Rust（loomgui_core + loomgui_ffi_c，csbindgen）、Unity 6.5 C#（Texture2D.LoadImage / File.ReadAllBytes）、blob 跨语言 SOA 契约。

```rust
//! 纹理注册表（spec §5.1）：src → TexMeta{tex_id, w, h}。per Stage。
//! core 只持整数 id + 维度；GPU 纹理由后端持有。tex_id 0 保留 = 未注册哨兵。

use std::collections::HashMap;

#[derive(Debug, Clone, Copy)]
pub struct TexMeta {
    pub tex_id: u32,
    pub width: u32,
    pub height: u32,
}

#[derive(Debug)]
pub struct TextureRegistry {
    map: HashMap<String, TexMeta>,
    next_id: u32, // 从 1 起；0 保留 = 未注册哨兵
}

impl Default for TextureRegistry {
    fn default() -> Self {
        Self { map: HashMap::new(), next_id: 1 }
    }
}

impl TextureRegistry {
    /// 注册：core 分配 tex_id（src 幂等，同 src 二次调用返回同 id、维度首次胜）。
    /// 返回 tex_id(>=1)。
    pub fn register(&mut self, src: &str, w: u32, h: u32) -> u32 {
        if let Some(m) = self.map.get(src) {
            return m.tex_id;
        }
        let id = self.next_id;
        self.next_id += 1;
        self.map.insert(src.to_string(), TexMeta { tex_id: id, width: w, height: h });
        id
    }

    pub fn get(&self, src: &str) -> Option<TexMeta> {
        self.map.get(src).copied()
    }

    /// 重载用：清表 + next_id 重置回 1。
    pub fn clear(&mut self) {
        self.map.clear();
        self.next_id = 1;
    }
}

#[cfg(test)]
mod tests {

    #[test]
    fn register_assigns_monotonic_ids_starting_at_1() {
        let mut r = TextureRegistry::default();
        assert_eq!(r.register("a.png", 10, 20), 1);
        assert_eq!(r.register("b.png", 30, 40), 2);
        assert_eq!(r.register("c.png", 50, 60), 3);
    }

    #[test]
    fn register_is_idempotent_on_src_first_dims_win() {
        let mut r = TextureRegistry::default();

        let m = r.get("a.png").unwrap();

    }
```

```rust
/// 非 UTF-8 src / null → 返回 0（哨兵，不 panic）。
#[no_mangle]
pub extern "C" fn loomgui_stage_register_texture(
    h: *mut StageHandle,
    src: *const u8,
    src_len: usize,
    width: u32,
    height: u32,
) -> u32 {
    if h.is_null() || src.is_null() { return 0; }
    let sh = unsafe { &mut *h };
    let bytes = unsafe { std::slice::from_raw_parts(src, src_len) };
    match std::str::from_utf8(bytes) {
        Ok(s) => sh.stage.textures.register(s, width, height),
        Err(_) => 0,
    }
}

/// collect：scene 中 Image src 去重后的数量（DFS 先序）。
#[no_mangle]
pub extern "C" fn loomgui_stage_image_src_count(h: *const StageHandle) -> usize {
    if h.is_null() { return 0; }
    let sh = unsafe { &*h };
    sh.stage.image_srcs().len()
}

/// collect：第 index 个 Image src 的 UTF-8 串指针；*out_len = 字节长。
/// OOB / null → null。串归 Stage 拥有，下次 load 前有效。
#[no_mangle]
pub extern "C" fn loomgui_stage_image_src_at(
    h: *const StageHandle,
    index: usize,
    out_len: *mut usize,
) -> *const u8 {
    if h.is_null() { return std::ptr::null(); }
    let sh = unsafe { &*h };
    let srcs = sh.stage.image_srcs();
    if index >= srcs.len() { return std::ptr::null(); }
```

```csharp
            var b = new List<byte>();

            b.AddRange(System.BitConverter.GetBytes(3u));           // version=3
            b.AddRange(System.BitConverter.GetBytes(1u));           // node_count=1

            const int HeaderLen = 12 + 14 * 4 + 2 * 4 + 2 * 4 + 2 * 4; // = 92

            b.AddRange(System.BitConverter.GetBytes(arenaOff));
            b.AddRange(System.BitConverter.GetBytes(arenaLen));

            }
            finally

                Object.DestroyImmediate(redTex);
                Object.DestroyImmediate(root);
```

```csharp
        /// v1b.2：collect scene 的 Image src → 读 StreamingAssets PNG → register → 建 _texMap。
        /// 缺图/坏图 → LogError 跳过（白占位）。FFI 调用 + 指针操作在 unsafe 块内（镜像 LateUpdate 模式）。
        void LoadTextures()
        {
            _texMap.Clear();
            nuint count;
            unsafe { count = Native.loomgui_stage_image_src_count(_stage); }
            for (nuint i = 0; i < count; i++)
            {
                byte* p;
                nuint len = 0;
                unsafe { p = Native.loomgui_stage_image_src_at(_stage, i, &len); }
                if (p == null) continue;
                string src = System.Text.Encoding.UTF8.GetString(p, (int)len);
                string path = System.IO.Path.Combine(Application.streamingAssetsPath, src);
                byte[] bytes;
                try { bytes = System.IO.File.ReadAllBytes(path); }
                catch (System.Exception e) { Debug.LogError($"[LoomGUI] texture not found: {src} ({e.Message})"); continue; }
                var tex = new Texture2D(2, 2);
                if (!tex.LoadImage(bytes)) { Debug.LogError($"[LoomGUI] bad png: {src}"); continue; }
                uint tid;
                unsafe { tid = Native.loomgui_stage_register_texture(_stage, p, len, (uint)tex.width, (uint)tex.height); }
                if (tid != 0) _texMap[tid] = tex;
                else { if (Application.isPlaying) Object.Destroy(tex); else Object.DestroyImmediate(tex); }
            }
        }
```

> 注：`Native.loomgui_stage_image_src_at` 返回的 `p` 直接复用作 register 的 src 指针（同一 UTF-8 串，stage 缓存有效期内）。csbindgen 映射：`*const u8→byte*`、`*mut usize→nuint*`、`usize→nuint`。T5 build.rs 重生成 `LoomGUIBindings.cs` 后核对 `Native` 类三函数签名与此用法一致（若 src 参数 csbindgen 生成 `byte*` + `nuint`，则上面 `(byte*)p`/`len` 直接可用）。

Awake `_usePackage` 分支（`LoadPackage()` 成功后）调 `LoadTextures()`；inline 分支不调（v1b.2 inline 仍白占位）。具体：在 `if (loaded) { ... }` 成功体里、`_mm = new MaterialManager(shader);` 之后加：

```csharp

```
