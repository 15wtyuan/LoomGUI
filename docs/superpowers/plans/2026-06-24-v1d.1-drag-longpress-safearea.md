# v1d.1 拖拽 + 长按 + safe-area Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 给 LoomGUI 加 drag（opt-in `draggable`，events-only 不跟手）+ longpress（universal，1.5s/50px）+ safe-area（根 letterbox），检测在 core，C# 路由，机制/阈值镜像 fgui。

**Architecture:** drag + longpress 都在 `loomgui_core/src/input.rs` 的 `PointerState` 状态机检测，emit 新 `EVT_DRAG_START/MOVE/END`(6/7/8) + `EVT_LONG_PRESS`(9)（EventRecord 仍 20B）。`Node` 加 `draggable` 字段（HTML `draggable="true"` 属性，pkg.bin round-trip）。C# `EventType` +4 值、`DispatchPending` 加 BubbleRoute 分支。safe-area 纯 Unity 侧 `LoomStage.ConfigureTransforms` + `LoomInputCollector` 对齐。

**Tech Stack:** Rust（loomgui_core + loomgui_ffi_c，taffy/bincode/csbindgen）、Unity 6.5 C#（URP，新旧输入系统兼容）。

## Global Constraints

- 版本串 `loomgui_version()` → `"v1d.1"`（T5）。
- ABI 不增字节：`PointerEvent` 16B、`EventRecord` 20B、`PointerKind` `#[repr(u8)]`。新事件复用 `event_type:u8` 空位 6/7/8/9，C# `LoomEvent`/`EventType` 加值不增字段。
- fgui 阈值（已核实源码）：drag `UIConfig.touchDragSensitivity=10` / `clickDragSensitivity=2`（per-axis OR，touch_id==-1→mouse=2，else→touch=10）；longpress `LongPressGesture.TRIGGER=1.5s` / `holdRangeRadius=50px`。drag 阈值 < click 容忍（click 10/50）。
- drag 阈值常量名：`DRAG_THRESHOLD_MOUSE: f32 = 2.0`、`DRAG_THRESHOLD_TOUCH: f32 = 10.0`。longpress：`LONGPRESS_TRIGGER: f32 = 1.5`、`LONGPRESS_RADIUS: f32 = 50.0`。
- drag **opt-in**（`Node.draggable`，HTML `draggable="true"` 属性）；longpress **universal**（无 flag，任何 down_node 按住即触发，无 listener 时 C# 跳过）。
- drag-start 置 `click_cancelled=true`（drag 必取消 click）；longpress **与 click 独立**（不自动取消，业务要消费调既有 `CancelTouch`）。
- drag 跟手（自动移动节点）→ **defer v1d.3**（transform 启用）。v1d.1 只发事件。
- Canceled 也发 `EVT_DRAG_END`（照 fgui onTouchEnd）。
- 两机串行：Rust 改（T1-T5）本机 `cargo test` 自检 + 重编 .dll commit（家里机才能验）；C#（T6-T8）本机写代码不编译，家里机 Unity 验。
- Ponytail：最短 diff，YAGNI，简化标 `ponytail:` 注释。time 用 f32。
- `disabled` 节点不发起 drag/longpress（照 click 抑制）；`touchable=false` 已被 hit_test 排除不会成 down_node。
- pkg.bin 加 `draggable`：**复用 NodeBlock 末尾加 1 字节 flags**（v3→v4，`MIN_VERSION=MAX_VERSION=4`）。改 pkg 格式必重编 .dll + commit（坑 10）。
- safe-area 默认 `_safeArea=true`（无刘海屏 `Screen.safeArea==全屏` 零回归）；相机 orthoSize 不变。

fn gather_rec(
    tree: &ElementTree,
    styles: &[ResolvedStyle],
    el_id: ElementId,
    parent_idx: Option<usize>,
    entries: &mut Vec<(Option<usize>, NodeKind, ResolvedStyle, Vec<String>, Option<String>, bool)>,
) -> usize {
    let el = &tree.nodes[el_id.0];
    let style = &styles[el_id.0];
    let kind = match el.tag.as_str() {
        "div" | "l-container" => NodeKind::Container,
        "button" => NodeKind::Button,
