# v1.2 border-radius Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让 div/button 容器支持 `border-radius`（1~4 值 + `h / v` 椭圆角 + px/%），圆角 mesh 三角扇几何在 Rust 核心生，零新 payload/shader/blob/mask。

**Architecture:** ResolvedStyle 加 `border_radius: BorderRadius`（存 CSS 原始值 px/%，渲染期 resolve）；mapping.rs 解析 `border-radius` arm + `parse_radius_group` helper；render/mesh.rs 加 `rounded_rect`（三角扇，照搬 fgui 自适应分段/末段精度锁/直角分支 + 两改进）；render/mod.rs Container/Button 分支 resolve + `all_zero` 分流（radius=0 走旧 quad 零回归）。PKG_FORMAT_VERSION 8→9。FFI/Unity 零改（重建 dll）。

**Tech Stack:** Rust（loomgui_core，taffy/cssparser/scraper）+ Unity C#（零改）+ HTML/CSS showcase。

## Global Constraints

- **围栏子集**：`border-radius: <px|%>{1,4} [ / <px|%>{1,4} ]?`。砍 longhand（`border-top-left-radius` 等）/ 圆角描边 / 圆角裁子节点 / 椭圆形状 / img 圆角（spec §0/§2）。
- **% 渲染期 resolve**：水平半径 = `width × pct/100`，垂直 = `height × pct/100`（width/height = layout rect w/h）。ResolvedStyle 存 CSS 原始值（`LengthPercentage`），**不在打包期 resolve**。
- **半径钳制（改进 1）**：CSS 按边缩放 `scale = min(1, w/(rx_TL+rx_TR), w/(rx_BR+rx_BL), h/(ry_TL+ry_BL), h/(ry_TR+ry_BR))`，所有半径 × scale + `max(0.0)` 防负。只缩不放。
- **角序（改进 2）**：TL→TR→BR→BL，起始角 TL=π / TR=-π/2 / BR=0 / BL=π/2。
- **照搬 fgui**：三角扇拓扑、自适应分段 `ceil(π·max(rx,ry)/8)+1`（最小 2）、末段精度锁 `start+π/2`、直角分支（`rx<=0||ry<=0` → 单顶点）。
- **零回归**：四角 (rx,ry) 全 0 → 走旧 `quad`（4 顶点），dirty hash 不变。
- **节点范围**：仅 div/button 容器（Container/Button 分支）；Image/Text 不加圆角。
- **共存**：圆角 + background-color + background-image（cover/contain/stretch）三共存，UV 线性映射到 `fit_uv` 子区。
- **PKG_FORMAT_VERSION 8→9**（坑 74：bincode 字段序变，旧 v8 pkg 须拒绝+重建）。
- **Rust build + commit + push 在本机**；Unity PlayMode 验收在家里机（改 Rust 后必重编+commit .dll）。
- **main 直推**。
- **Ponytail 全开**：最短差异，YAGNI，无未请求抽象。
- **不建新 payload 变体 / 不改 shader / 不改 FFI blob SOA / 不改 mask 机制 / 不改 NodeAnim / 不改打包器收集链 / 不改 dirty.rs 实现**（hash 已含 verts.len）。

```rust
/// 解析 border-radius 1~4 值（每值 px 或 %）→ [LengthPercentage;4]（TL,TR,BR,BL）。
/// 与 parse_four 同序，但保留 %。任一值非法（auto/inherit/initial/非 px-% 数字）→ None
/// （CSS：整条声明无效）。
fn parse_radius_group(s: &str) -> Option<[LengthPercentage; 4]> {
    let parts: Vec<&str> = s.split_whitespace().collect();
    let p = |i: usize| -> Option<LengthPercentage> {
        let tok = parts.get(i)?.trim();
        if tok == "auto" || tok == "inherit" || tok == "initial" {
            return None;
        }
        Some(parse_lp(tok))
    };
    let v0 = p(0)?;
    Some(match parts.len() {
        1 => [v0, v0, v0, v0],
        2 => [v0, p(1)?, v0, p(1)?],
        3 => [v0, p(1)?, p(2)?, p(1)?],
        _ => [v0, p(1)?, p(2)?, p(3)?],
    })
}
```

在 `mesh.rs` 的 `quad` 函数之后（约第 40 行后），加 `rounded_rect`：

```rust
/// 生成圆角矩形的 verts/uvs/colors/indices（SOA 四表，与 quad 同形）。
///
/// - 三角扇：中心点 + 4 角圆弧顶点，三角形 (0, i, i+1) 连接，末尾回 1 闭合。
/// - radii = [TL, TR, BR, BL]，每角 (h, v) 像素半径（本函数内钳制）。
/// - UV 线性映射：顶点归一化位置 (pos-rect.xy)/rect.size × (uv_max-uv_min) + uv_min。
///   与 fit_uv 共存：uv_min/uv_max 即 fit_uv 算出的子区。
/// - 产 design y-down 坐标；v 翻转由调用点交换 uv v 处理（同 quad）。
/// - 退化（w/h≤0）走 quad fallback。
#[allow(clippy::type_complexity)]
pub fn rounded_rect(
    rect: &Rect,
    color: [f32; 4],
    radii: &[(f32, f32); 4],
    uv_min: [f32; 2],
    uv_max: [f32; 2],
) -> (Vec<[f32; 2]>, Vec<[f32; 2]>, Vec<[f32; 4]>, Vec<u32>) {
    let (w, h) = (rect.w, rect.h);
    if w <= 0.0 || h <= 0.0 {
        return quad(rect, color, uv_min, uv_max);
    }
    // 改进 1：CSS 按边缩放钳制（vs fgui per-corner min）。两邻角半径和不超过边长，等比缩放；
    // 只缩不放（min(1.0) 兜底）；防负 max(0.0)。
    let (tl, tr, br, bl) = (radii[0], radii[1], radii[2], radii[3]);
    let scale = 1.0_f32
        .min(w / (tl.0 + tr.0).max(1e-6))
        .min(w / (bl.0 + br.0).max(1e-6))
        .min(h / (tl.1 + bl.1).max(1e-6))
        .min(h / (tr.1 + br.1).max(1e-6))
        .min(1.0);
    let clamp_r = |r: (f32, f32)| -> (f32, f32) {
        ((r.0 * scale).max(0.0), (r.1 * scale).max(0.0))
    };
    let tl = clamp_r(tl);
    let tr = clamp_r(tr);
    let br = clamp_r(br);
    let bl = clamp_r(bl);

    let lerp = |a: f32, b: f32, t: f32| a + (b - a) * t;
    let (umin, vmin) = (uv_min[0], uv_min[1]);
    let (umax, vmax) = (uv_max[0], uv_max[1]);

    let mut verts: Vec<[f32; 2]> = Vec::new();
    let mut uvs: Vec<[f32; 2]> = Vec::new();
    let mut colors: Vec<[f32; 4]> = Vec::new();
    // 中心点（索引 0）
    let cx = rect.x + w / 2.0;
    let cy = rect.y + h / 2.0;
    verts.push([cx, cy]);
    uvs.push([lerp(umin, umax, 0.5), lerp(vmin, vmax, 0.5)]);
    colors.push(color);

    // 改进 2：角序 TL→TR→BR→BL（CSS 视觉序）。
    // 起始角 TL=π, TR=-π/2, BR=0, BL=π/2（逆时针 design y-down）。圆心 = 角顶点内缩 (rx,ry)。
    let corners: [(f32, f32, [f32; 2], f32); 4] = [
        (tl.0, tl.1, [rect.x + tl.0,         rect.y + tl.1],         std::f32::consts::PI),
        (tr.0, tr.1, [rect.x + w - tr.0,     rect.y + tr.1],         -std::f32::consts::FRAC_PI_2),
        (br.0, br.1, [rect.x + w - br.0,     rect.y + h - br.1],     0.0),
        (bl.0, bl.1, [rect.x + bl.0,         rect.y + h - bl.1],     std::f32::consts::FRAC_PI_2),
    ];
    for (rx, ry, center, start) in corners {
        if rx <= 0.0 || ry <= 0.0 {
            // 直角：单顶点 = 角顶点（照 fgui radius==0 分支）。
            // 角顶点 = 圆心 ± (rx,ry) 方向退到 rect 角；rx/ry 已 0 → 圆心即角顶点偏移 0 = rect 角。
            // 按 start 方向定符号：起始边方向 = (cos(start), sin(start))。
            let px = center[0] + start.cos() * rx;
            let py = center[1] + start.sin() * ry;
            verts.push([px, py]);
            uvs.push([lerp(umin, umax, (px - rect.x) / w), lerp(vmin, vmax, (py - rect.y) / h)]);
            colors.push(color);
            continue;
        }
        // 自适应分段（照搬 fgui：ceil(π·max(rx,ry)/8)+1，最小 2）
        let sides = ((std::f32::consts::PI * rx.max(ry) / 8.0).ceil() as i32 + 1).max(2);
        let delta = std::f32::consts::FRAC_PI_2 / sides as f32;
        for j in 0..=sides {
            let a = if j == sides {
                start + std::f32::consts::FRAC_PI_2  // 末段精度锁（照 fgui）
            } else {
                start + delta * j as f32
            };
            let px = center[0] + a.cos() * rx;
            let py = center[1] + a.sin() * ry;
            verts.push([px, py]);
            uvs.push([lerp(umin, umax, (px - rect.x) / w), lerp(vmin, vmax, (py - rect.y) / h)]);
            colors.push(color);
        }
    }
    // 三角扇：(0, i, i+1)，末尾回 1 闭合
    let n = verts.len() as u32;
    let mut indices: Vec<u32> = Vec::new();
    for i in 1..n {
        let next = if i + 1 < n { i + 1 } else { 1 };
        indices.extend_from_slice(&[0, i, next]);
    }
    (verts, uvs, colors, indices)
}
```
