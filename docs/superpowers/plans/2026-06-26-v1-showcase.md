# v1 Showcase 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 建一个 fgui 式交互功能 showcase（8 区 46 卡），整合 v1 全部已实现功能，供家里机 PlayMode 逐区点验。

**Architecture:** 纯 sample（HTML/CSS DSL + C# 反馈脚本），零 core/FFI 改动。单页长滚动 + 顶部固定 nav（点 nav → SetScrollPos 跳区）。交互反馈用 CSS 伪类 + tween 灯阵（v1 无运行时改文本 API）。

**Tech Stack:** LoomGUI HTML-as-DSL（围栏 v1 子集）+ loomgui_pkg 打包器 + Unity 6.5 C#（LoomStage + LoomEventHandler + tween API）。

## Global Constraints

[从 spec `docs/superpowers/specs/2026-06-26-v1-showcase-design.md` 抄，每任务隐含遵守。]

- **仅用围栏内属性**（v1-scope §2）：`display:flex/none`、`flex-direction/justify-content/align-items/align-self/flex*/gap/order`、`width/height/min*/max*/aspect-ratio`（px/%/auto）、`padding/margin`、`border`(color/width)、`position:relative`、`background-color`、`opacity`、`overflow/overflow-x/overflow-y`(visible/hidden/scroll/auto)、`color/font-size/font-family/font-weight/letter-spacing/text-align/line-height/white-space`（**font-style 是 gap，标灰 skip**）、`transform`(translate/rotate/scale)、`<img src>`、`draggable/tabindex/id/class` 属性、`:hover/:active/:focus` 伪类。
- **禁用围栏外属性**（v1.x）：`border-radius`/渐变/`box-shadow`/`filter`/`clip-path`/九宫格/`position:absolute/sticky/fixed`/`z-index`/`grid`/`background-image`/`background-size`/`font-style`/CSS `transition`。**gap 项**（background-image/background-size/font-style）：卡内标灰 skip 文本说明，不写该属性。
- **零 core/FFI/blob/pkg 契约改动**：不改 Rust、不重编 .dll、不改 blob/pkg 版本（version 仍 `v1e`）。
- **反馈不依赖运行时改文本**：v1 无 set_text/set_style API；计数用 C# 内存 int + 灯阵 tween 点亮（`TweenProp.Opacity` 0→1）。hover/active/disabled/focus 用 CSS 伪类。
- **节点数 <500**（v1 性能线）：46 卡 × ~7 ≈ 350。
- **设计稿 1080×1920**（竖屏，参考分辨率缩放 + safe-area）。
- **CSS 外部文件**：HTML 无 `<style>`/`<link>`；CSS 单行紧凑写法（`.x{...}`，与现有 sample 一致）。v1 的 div 默认 `flex-direction:column`（横排要显式 `flex-direction:row`）。
- **落点**（按 v1d3/v1d4 惯例，spec §9 的 `loomgui_pkg/samples/` 调整为 Unity Samples/）：
  - HTML/CSS 源 + 图标：`loomgui_unity/Assets/LoomGUI/Samples/v1-showcase/`（index.html + style.css + icons/*.png）
  - C# driver：`loomgui_unity/Assets/LoomGUI/Runtime/LoomShowcaseDriver.cs`
  - 字体：`loomgui_unity/Assets/LoomGUI/Fonts/Sarasaui-Regular.ttf`（含 CJK）
  - pkg 产物：`loomgui_unity/Assets/StreamingAssets/loom_showcase.pkg.bin` + `loom.atlas.png`
  - 场景：复用 `loomgui_unity/Assets/Scenes/SampleScene.unity`（加 LoomStage GO）
- **打包命令**：`cargo run -p loomgui_pkg -- loomgui_unity/Assets/LoomGUI/Samples/v1-showcase/index.html loomgui_unity/Assets/LoomGUI/Samples/v1-showcase/style.css -o loomgui_unity/Assets/StreamingAssets/loom_showcase.pkg.bin -w 1080 -h 1920`
- **素材许可**：图标 CC0/MIT（Lucide/Tabler/Kenney 任选），字体 OFL（Sarasa）。均在 plan/仓库标注来源。
- **两台机**：本机编码 + 打包 + commit；家里机纯 PlayMode 视觉验收（本机无 Unity 跑）。
- **每任务验证 = 打包成功 + 围栏过 + `cargo test --workspace` 不回归**（383 测仍绿）。视觉/交互验收在 plan 末尾 acceptance checklist，家里机执行。

```csharp
using UnityEngine;

namespace LoomGUI
{
    // v1 showcase driver: nav 跳转 + 启动错峰入场 + 交互灯阵（T7）+ tween 演示（T8）。
    public unsafe class LoomShowcaseDriver : MonoBehaviour
    {
        [SerializeField] LoomStage _stage;
        [SerializeField] float[] _sectionY = { 0f, 700f, 1500f, 2300f, 3000f, 3800f, 4500f, 5200f }; // 设计期累积高度（sec-1..sec-8 顶部 y）

        uint _scrollNode = uint.MaxValue;
        uint[] _navNodes = new uint[8];

        void Awake()
        {
            if (_stage == null) _stage = GetComponent<LoomStage>();
            if (_stage == null) { Debug.LogError("[Showcase] 无 LoomStage"); return; }
        }

        void Start()
        {
            _scrollNode = _stage.FindNodeById("main-scroll");
            for (int i = 0; i < 8; i++)
            {
                _navNodes[i] = _stage.FindNodeById("nav-" + (i + 1));
                if (_navNodes[i] != uint.MaxValue)
                    _stage.EventHandler.AddListener(_navNodes[i], EventType.Click, OnNavClick);
            }
            Debug.Log($"[Showcase] scroll={_scrollNode} nav0={_navNodes[0]}（点 nav 跳区）");
            StaggeredEntrance();
        }

        // design-taste §4: 启动错峰入场。各 sec 卡 tween opacity 0→1 + delay 递增。
        // 注：HTML 各 sec 初始 opacity:0（见 style.css .sec），靠 tween 拉亮；同时验证 §7.1 opacity + §7.3 delay。
        void StaggeredEntrance()
        {
            for (int i = 0; i < 8; i++)
            {
                uint node = _stage.FindNodeById("sec-" + (i + 1));
                if (node == uint.MaxValue) continue;
                _stage.Tween(node, TweenProp.Opacity,
                    new float[] { 0f, 0, 0, 0 }, new float[] { 1f, 0, 0, 0 },
                    0.4f, Ease.CubicOut, i * 0.04f, 0);
            }
        }

        void OnNavClick(EventContext ctx)
        {
            if (_scrollNode == uint.MaxValue) return;
            for (int i = 0; i < 8; i++)
            {
                if (ctx.target == _navNodes[i])
                {
                    _stage.SetScrollPos(_scrollNode, 0f, _sectionY[i], true);
                    return;
                }
            }
        }
    }
}
```
